ArcSight CEF
Common Event Format (CEF) is an extensible, text-based format designed to support
        multiple device types by offering the most relevant information.  
The Common Event Format (CEF) standard format, developed by ArcSight, enables vendors and
            their customers to quickly integrate their product information into ESM.
