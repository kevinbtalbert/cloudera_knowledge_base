Opening the Alerts User
      Interface
You can use the Alerts user interface to display, filter, and sort events and their
      associated fields. You can also use the UI to escalate, add comments to, and group
      events.

From the Ambari Dashboard navigation panel, click Metron.
            

Verify that the Summary tab is selected. 

From the Quick Links menu, select Alerts
                  UI. 
The Alerts UI tool displays in a separate browser tab.
