Data Services and Integration Layer
The data services and integration layer is a set of three CCP modules that provides
      different features for different SOC personas. 
CCP provides three modules for the integration layer.


Security data vault

Stores the data in HDFS.

Search portal

The Metron dashboard.

Provisioning, management, and monitoring tool

A CCP-provided management module that expedites provisioning and managing
                     sensors. Other provisioning, management, and monitoring functions are supported
                     through Apache Ambari.
