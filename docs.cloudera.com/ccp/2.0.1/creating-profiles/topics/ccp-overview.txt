Introduction to CCP
      Analytics
Data Scientists are frequently responsible for performing data science life cycle
      activities, including training, evaluating, and scoring analytical models. Cloudera
      Cybersecurity Platform (CCP) provides the ability to create profiles and models, analyze data
      using statistical and mathematical functions and Apache Zeppelin, and create runbooks for SOC
      analysts and investigators.
