Managing Sensors
You can manage your sensors and associated topologies using either the Hortonworks
      Cybersecurity Platform (HCP) Management user interface or the Apache Storm UI. The following
      procedures use the HCP Management UI to manage sensors. For information about using Storm to
      manage sensors, see the Storm documentation.
Start a SensorAfter you install a sensor, you can start it using Management user       interface.Stop a SensorAfter you install a sensor, you can stop it using the Management user       interface.Modify a SensorYou can modify any sensor listed in Cloudera Cybersecurity Platform (CCP) Management       user interface.Delete a SensorYou can delete a sensor if you don't need it.Parent topic: Managing Overview
