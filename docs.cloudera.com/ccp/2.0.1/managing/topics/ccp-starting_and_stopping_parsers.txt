Start and Stop
      Parsers
You might want to stop or restart parsers as you refine your cybersecurity monitoring.
      You can easily stop and start parsers by using Ambari.

 Display the Ambari UI and navigate to Services >
                  Metron > SUMMARY:






 Click METRON PARSERS to display the
                  Components window.
The Components window displays a list of Metron hosts and which components reside
               on each host.

               box.
Ambari displays the Background Operations dialog box which
               provides the status of the operation.

Click OK to exit the Background
                  Operations dialog box.
