Technical Preview Features
Some features included in the CCP 2.0.1 release are not yet officially supported by
      Hortonworks. These technical preview features are still under development and are not
      recommended for a production environment. 

Table 1. Technical Preview Features

Feature
Description



Event time profiling 
Changes the behavioral profiling window to use the event time instead
                           of system time. This better reflects the actual timing of the event and
                           increases the accuracy of the profiles.
