Configure the Metron
      Dashboard to View the New Data Source Telemetry Events
After Cloudera Cybersecurity Platform (CCP) is configured to parse, index, and persist
      telemetry events and NiFi is pushing data to CCP, you can view streaming telemetry data in the
      Metron Dashboard.
