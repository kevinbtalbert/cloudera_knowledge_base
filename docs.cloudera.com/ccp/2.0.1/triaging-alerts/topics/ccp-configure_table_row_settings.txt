Modify the Alert Data
    Refresh Rate
You can use the table row settings in the Alerts table to change the refresh rate of
    the alert data.

 Click the  (slides icon) at the top of the table to display the Settings
          dialog box.  






 To modify the rate at which the Alerts table is refreshed with new alert information,
          choose a value under Refresh Rate. 
After you modify the refresh rate, you will see a warning indicating that the current
          alert data is not in sync with your current parameters.
Click the 
          (search) button to refresh the alerts data.
