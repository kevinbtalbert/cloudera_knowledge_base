Modify Number of Alert
    Table Rows
You can use the table row settings in the Alerts table to modify the number of rows
    that appear in the Alerts table.

 Click the  (slides icon) at the top of the table to display the Settings
          dialog box.  






 To modify the number of rows displayed in the Alerts table, choose a value under
            Rows Per Page.  

noteThe number of rows that are visible in the Alerts table is restricted by the size of
            your browser window.
