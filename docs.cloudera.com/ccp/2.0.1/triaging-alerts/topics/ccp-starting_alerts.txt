Start and Pause Automatic
      Polling
The automatic polling in the Alerts UI defaults to a paused state when you first log
      in. To start automatic polling, you must click the play button.
 To start automatic polling, click the  (play)
               button.
Polling is also paused whenever you open any configuration panels or use the
                  Searches field.

To manually pause automatic polling, click the  (pause)
               button.
