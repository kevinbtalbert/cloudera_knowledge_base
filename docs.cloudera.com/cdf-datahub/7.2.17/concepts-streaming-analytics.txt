Cloudera Streaming Analytics


Cloudera Streaming Analytics powered by Apache Flink offers a framework
for real-time stream processing and streaming analytics. With Flink
implemented on CDP Private Cloud Base, you will be able to use the
built-in connectors of Runtime components, cluster and service
management with Cloudera Manager. Additionally, Streaming Analytics in
Cloudera provides an integrated job management interface, SQL Stream
Builder, to create streaming applications using SQL.
