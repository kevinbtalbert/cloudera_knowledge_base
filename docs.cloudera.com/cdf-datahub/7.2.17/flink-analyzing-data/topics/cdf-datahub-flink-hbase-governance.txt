Metadata governance with Atlas
You can use Atlas to find, organize and manage different assets of data about your
        Flink applications and how they relate to each other. This enables a range of data
        stewardship and regulatory compliance use cases.

Go to Management Console > Data Lakes.

Search for your environment from the list of available environments.

Select Atlas from the list of services.

Search and select flink_application  from the Search By Type bar.

Click the Name of your application.

Select Properties, Lineage, Relationships, Classifications or Audits tabs for
                    more information about your application.
