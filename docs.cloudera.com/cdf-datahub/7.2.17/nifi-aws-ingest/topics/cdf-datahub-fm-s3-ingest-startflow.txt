Start the data flow
When your flow is ready, you can begin ingesting data into Amazon S3 buckets. Learn
    how to start your S3 ingest data flow.

To initiate your data flow, select all the data flow components you want to
          start.Click the Start icon in the Actions 
          toolbar.Alternatively, right-click a single component and choose
            Start from the context menu. Data should be read from Kafka and
          it should be written to the defined folder of your AWS S3 bucket



It is useful to check that data is running through the flow you have
      created.
