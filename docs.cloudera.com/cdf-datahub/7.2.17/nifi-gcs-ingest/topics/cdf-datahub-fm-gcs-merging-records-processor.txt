Configure the processor for merging records
You can use merge together multiple record-oriented flow files into a large flow file
    that contains all records of your data input. Learn how you can configure the
      MergeContent processor for your Google Cloud Storage (GCS) ingest data
    flow.

Launch the Configure Processor window, by right clicking the
            MergeContent processor and selecting
          Configure.This gives you a configuration dialog with
          the following tabs: Settings, Scheduling,
            Properties, Comments.Configure the processor according to the behavior you expect in your data
          flow.When you have finished configuring the options you need, save the changes by
          clicking Apply.
Make sure that you set all required properties, as you cannot start the processor until
            all mandatory properties have been configured.
For a complete list of MergeContent properties, see the
              processor documentation.
