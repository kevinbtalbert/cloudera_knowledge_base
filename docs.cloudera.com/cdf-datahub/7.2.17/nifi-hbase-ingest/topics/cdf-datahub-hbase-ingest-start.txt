Start your data flow
Start your data flow to verify that you have created a working dataflow and to begin
        your data ingest process. 

To initiate your data flow, select all the flow components you want to
                    start.

Click the Start icon in the
                        Actions toolbar. 
Alternately, right-click a single component and choose
                        Start from the context menu.
