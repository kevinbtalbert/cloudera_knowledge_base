Setting workload password
Learn how to set your personal workload password.
You must have created a Machine User account.

Click Profile in your user menu on the CDP
                    console.






On your profile page, click Set Workload Password and
                    follow the prompts to set a password for your user. You can leave the
                        Environment field empty.
