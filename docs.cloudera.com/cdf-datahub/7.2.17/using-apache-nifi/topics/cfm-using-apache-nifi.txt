Using Apache NiFi
To access the Apache NiFi User Guide, see the Apache NiFi
  documentation.

importantThe Apache NiFi documentation is available both online and in the NiFi UI.
    The online documentation always shows the latest version of the NiFi documentation. There may be
    differences in the content compared to the version of NiFi you are using. You can access the
    documentation corresponding to the NiFi version you are using directly from the NiFi UI.
Launch the NiFi UI.
From the Global Menu, select Help >  Documentation.
