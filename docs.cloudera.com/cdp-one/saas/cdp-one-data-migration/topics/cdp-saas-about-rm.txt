Introduction to Replication Manager
Replication Manager is a service to copy and migrate data from CDH 5.13+ version and
        above clusters to CDP One endpoint. The currently supported
        Public Cloud service is Amazon S3.
Accessing the Replication Manager serviceYou can access the Replication Manager service by logging into Cloudera Data         Platform.Parent topic: Migrating HDFS and Hive data from CDH to CDP One
