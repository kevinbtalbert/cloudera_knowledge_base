Accessing the Spark History Server
You can access the Spark History Server for your Spark cluster
    from the Cloudera Data Platform (CDP) Management Console interface. 
In the Management Console, navigate to your Spark cluster
              (Data Hub
              Clusters > <Cluster
            Name>).Select the Gateway tab.Click the URL for Spark History
          Server.
