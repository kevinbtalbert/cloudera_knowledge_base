Monitoring and Debugging Spark Applications

 To obtain information about Spark application behavior you can consult
      YARN logs and the Spark web application UI. These two methods provide
      complementary information. For information how to view logs created by
      Spark applications and the Spark web application UI, see Monitoring Spark Applications.
