Updating a notifier
Learn how to use Streams Messaging Manager (SMM) to update a notifier that you create
        in your environment.

Perform the following steps to update a notifier:


From the left navigation pane, select Alerts.
The Alerts Overview page appears.

Click NOTIFIERS.

Find the notifier you want to update from the list of available notifiers, and
                    click the pencil icon beside the notifier.
The Notifier window appears.

Edit the properties.

Click Save.
