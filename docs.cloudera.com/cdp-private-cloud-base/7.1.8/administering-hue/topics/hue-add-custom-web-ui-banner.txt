Adding a custom banner
You can add a custom banner to the Hue web interface by adding your
      custom HTML to the Top Banner Custom HTML property in Cloudera Manager.

Log in to Cloudera Manager as an Administrator.

Go to Clusters > Hue service > Configuration and add the banner test in the Top Banner Custom
            HTML field as follows:
<H1>Your company's custom Hue Web UI
          banner</H1>

Click Save Changes.

Restart the Hue service.
