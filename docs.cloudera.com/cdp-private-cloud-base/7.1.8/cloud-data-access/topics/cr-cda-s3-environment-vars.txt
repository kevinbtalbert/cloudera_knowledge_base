Using Environment Variables to Authenticate
AWS CLI supports authentication through environment variables. These same environment variables will be used by Hadoop
    if no configuration properties are set.

The environment variables are:


Environment Variable
Description



AWS_ACCESS_KEY_ID
Access key


AWS_SECRET_ACCESS_KEY
Secret key


AWS_SESSION_TOKEN
Session token (only if using session authentication)
