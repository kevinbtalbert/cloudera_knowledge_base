Working with Google Cloud Storage
Cloudera Data Platform Data Centre allows you to configure access from a cluster to
      Google Cloud Storage (GCS) in order to access cloud data.
Configuring Access to Google Cloud StorageAccess to Google Cloud Storage can be configured separately for each cluster by             providing the service account email address.
