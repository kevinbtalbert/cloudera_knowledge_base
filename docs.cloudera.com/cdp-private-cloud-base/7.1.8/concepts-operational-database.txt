Operational Database


Cloudera Operational Database is powered by Apache HBase and Apache
Phoenix. Apache HBase is a NoSQL database that provides real-time
read/write access to large datasets. Apache Phoenix provides a SQL
interface for managing data in Apache HBase. Operational Database
powered by Apache Accumulo is the Cloudera package of Apache Accumulo.
