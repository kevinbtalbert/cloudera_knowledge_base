Accessing the Cloudera Manager Admin Console
How to open and log in to the Cloudera Manager Admin
    Console.
To access the Cloudera Manager Admin Console:
Open the Cloudera Management Console in a Web browser using the
          following URL: 
http://<Cloudera Manager Server URL>:7180
Enter your user name.Enter your password.Click the Sign In button.
The Cloudera Manager Admin Console opens.
