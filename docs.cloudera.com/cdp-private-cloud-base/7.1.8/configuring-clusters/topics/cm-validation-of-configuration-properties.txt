Validation of Configuration Properties

Cloudera Manager validates the values you specify for
      configuration properties. If you specify a value that is outside the
      recommended range of values or is invalid, Cloudera Manager displays a
      warning at the top of the Configuration
      tab and in the text box after you click Save
        Changes. The warning is yellow if the value is outside the
      recommended range of values and red if the value is invalid.
