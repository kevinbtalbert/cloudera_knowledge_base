Viewing a List of All Suppressed Validations


Do one of the following:
From the Home page or the
              Status
            page of a cluster, select Configuration > Suppressed Health
                and Configuration Issues. 
From the Status page of a
            service, select Configuration > Category > Suppressions and select Status > Non-default.
From the left menu, select Hosts > Hosts
                Configuration, then Category > Suppressions
            and select Status > Non-default.
