Configure the scanner heartbeat using Cloudera Manager
You can configure the HBase scanner heartbeat using Cloudera Manager. Go to the HBase service.
Click the Configuration tab.
Select HBase or HBase
              Service-Wide.Locate the RPC Timeout property by typing its name in the Search
          box, and edit the property.Locate the HBase RegionServer Lease Period property by typing its
          name in the Search box, and edit the property.
Enter a Reason for change, and then click Save
            Changes to commit the changes.

Restart the role.

Restart the service.
