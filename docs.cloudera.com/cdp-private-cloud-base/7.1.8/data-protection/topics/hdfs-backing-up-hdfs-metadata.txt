Backing up HDFS metadata
HDFS metadata represents the structure and attributes of HDFS directories and files in
        a tree. You can back up the metadata without affecting NameNode availability. 

Introduction to HDFS metadata files and directoriesHDFS metadata represents the structure of HDFS directories and files in a tree.
