Delete snapshots using Cloudera Manager
You can use Cloudera Manager select from a snapshot configured for a directory and
        delete it.


From the Clusters tab, select the HDFS
                    service.

Go to the File Browser tab.

Go to the directory with the snapshot you want to delete.

In the list of snapshots, locate the snapshot you want to delete and click
                        .
