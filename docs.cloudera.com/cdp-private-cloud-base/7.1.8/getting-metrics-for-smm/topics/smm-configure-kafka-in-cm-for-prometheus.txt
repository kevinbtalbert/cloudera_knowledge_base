Kafka property configuration in Cloudera Manager for Prometheus
Learn about the Kafka properties that you need to configure in Cloudera Manager before
          you start using the Prometheus metric store.
Configure the following Kafka property in Cloudera Manager:
kafka.http.metrics.portUsed for exposing Cloudera Manager metrics.
            This port is now also shared to expose Prometheus metrics for Kafka.
