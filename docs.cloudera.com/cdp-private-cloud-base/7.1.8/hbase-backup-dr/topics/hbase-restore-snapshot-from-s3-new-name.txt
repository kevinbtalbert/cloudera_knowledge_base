Restore an HBase snapshot from Amazon S3 with a new name
By restoring an HBase snapshot stored in Amazon S3 with a new name, you clone the
        table without affecting the existing table in HBase. 
To clone the table without affecting the existing table in HBase, perform the
            following steps

Select the table in the Table Browser.

Click Restore Table From Snapshot As.

In the Restore As dialog box, enter a new
                    name for the table in the Restore As
                    field.

Select Remote S3 and choose the snapshot in
                    the list of available Amazon S3 snapshots.
