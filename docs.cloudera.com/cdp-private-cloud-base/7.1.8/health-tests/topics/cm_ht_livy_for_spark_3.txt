Livy for Spark 3 Livy Server for Spark 3 Health

 This Livy for Spark 3 service-level health test checks for the presence of a running, healthy Livy Server for Spark 3. The test returns "Bad" health if the service is running and the Livy Server for Spark 3 is not running. In all other cases it returns the health of the Livy Server for Spark 3. A failure of this health test indicates a stopped or unhealthy Livy Server for Spark 3. Check the status of the Livy Server for Spark 3 for more information. This test can be enabled or disabled using the Livy Server for Spark 3 Role Health Test Livy Server for Spark 3 service-wide monitoring setting.
