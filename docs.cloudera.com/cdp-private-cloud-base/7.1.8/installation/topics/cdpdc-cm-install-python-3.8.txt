Installing Python 3.8 for Cloudera Manager 7.7.3
Python 3.8 is required for using Cloudera Manager 7.7.3. Cloudera Manager 7.7.3 is
    supported only on Python 3.8. You must install Python 3.8 on all cluster hosts before installing
    Cloudera Manager and adding services to your cluster. 

Cloudera Manager 7.7.3-CHF4 is supported only on the following operating systems: 
RHEL 8.4
RHEL 8.6
RHEL 7.9 
SLES 15 SP4
