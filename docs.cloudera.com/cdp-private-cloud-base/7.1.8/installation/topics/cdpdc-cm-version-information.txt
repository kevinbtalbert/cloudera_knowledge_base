Cloudera Manager Version Information
You must choose the correct Cloudera Manager for your deployment
    This page provides a reference of Cloudera Manager versions, their release
    dates, and important compatibility information. 
Cloudera Manager 7.7.3 is an alternative to using Cloudera
      Manager 7.7.1 that provides Python 3 support on RHEL 8.4 and 8.6.
Cloudera Manager 7.7.3-CHF1 is an alternative to using Cloudera
      Manager 7.7.1 that provides Python 3 support on RHEL 8 .4, 8.6, and
      7.9.
importantYou must use Cloudera Manager
      7.7.3 or 7.7.3-CHF1 only when you need to use Python 3.8 for the Cloudera
      Manager agents. You must install Python 3.8 on all hosts before installing
      or upgrading to Cloudera Manager 7.7.3-CHF1. Cloudera Manager 7.7.3-CHF1
      is only supported with RHEL 8.4, 8.6, and 7.9. Python 2.7 is not
      supported. For more information, see the CDP Private Cloud Base Installation
        Guide.
noteCloudera Manager versions 7.5.1 and 7.5.4 are not compatible with
      Spark 3 CDS. You can use Cloudera Manager 7.4.4 if you are not using CDP Private Cloud Data Services but are
      planning to run Spark 3 CDS.
