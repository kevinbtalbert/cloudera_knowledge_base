Configuring Cloudera Manager to Use an Internal Remote Parcel Repository
In Cloudera Manager's parcel settings, add a path to the internal parcel
    repository.

Use one of the following methods to open the parcel settings page:

Navigation bar:
Click the parcel icon in the left navigation bar or click
                  Hosts and click the Parcels
                tab.
Click the Configuration button.

Menu:
Select
                    Administration > Settings.
Select
                  Category > Parcels.


Enter the path to the parcel. For example:
              http://<web_server>/cloudera-parcels/cdh7/7.0.3.1/
