Creating a Permanent Internal Repository
The following sections describe how to create a permanent internal repository using
    Apache HTTP Server.
Setting Up a Web ServerTo host an internal repository, you must install or use an existing Web server on an     internal host that is reachable by the Cloudera Manager host, and then download the repository     files to the Web server host.
