Step 4: Install Python 3.8 for Hue
Certain services, such as Hue, in CDP 7.1.8 and higher use Python 3.8. You must
        install Python 3.8 on all the hosts running the affected services after you have installed
        Cloudera Manager and before adding the services to your cluster.
noteInstalling Python 3.8 is mandatory if you want to use Hue.
Ubuntu 20 comes preinstalled with Python 3.8. You must install Python 3.8 manually on
            CentOS 7, RHEL 8, SLES 12, and Ubuntu 18.
