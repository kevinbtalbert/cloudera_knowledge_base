      driver for connecting to CDP Private Cloud Base
You learn how to download the Cloudera ODBC drivers for Hive and Impala.

Download the latest Hive ODBC driver for CDP from the
                  Cloudera ODBC driver download page. 

Go to the Impala ODBC driver page, and download the
               latest Impala ODBC driver.

Follow ODBC driver installation instructions on the download page.
