Maven artifacts
The following Maven <dependency> element is valid for the Apache Kudu GA release: 
<dependency>
  <groupId>org.apache.kudu</groupId>
  <artifactId>kudu-client</artifactId>
  <version><kudu-cdp-version></version>
</dependency>
For <kudu-cdp-version>, check Cloudera Runtime component
            versions in Release Notes.
Convenience binary artifacts for the Java client and various Java integrations (e.g.
         Spark, Flume) are also now available via the ASF Maven repository and the
            Central Maven repository.
