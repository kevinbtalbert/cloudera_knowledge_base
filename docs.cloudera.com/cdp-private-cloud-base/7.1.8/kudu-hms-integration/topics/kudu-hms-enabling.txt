Enabling the Hive Metastore integration
You can enable the Hive Metastore (HMS) integration with Kudu using Cloudera
        Manager.

 Upgrade any tables that may exist in Kudu's or in the HMS's catalog. For more
                information, see Upgrading Existing Tables.


In Cloudera Manager navigate to Kudu >  Configuration.

Find the HMS Service property and select the Hive
                    Metastore with which Kudu will synchronize its system catalog.
