Run a tablet rebalancing tool in Cloudera Manager
You access and run the tablet rebalancing tool from Cloudera Manager.
 Browse to Clusters > Kudu.

 Click Actions and select Run Kudu Rebalancer
            Tool. 

In Cloudera Manager, the rebalancer runs with the default
            flags.
