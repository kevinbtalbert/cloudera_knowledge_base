Deleting a Host from Cloudera Manager
To delete a host from Cloudera Manager, first decommission the host and then remove
    it.

Minimum Required Role:
Full
            Administrator. This feature is not available when using Cloudera
            Manager to manage Data Hub clusters.


In the Cloudera Manager Admin Console, go to Hosts > All Hosts.Select the hosts to delete.Select Actions for Selected > Hosts
              Decommission. Stop the Agent on the host.In the Cloudera Manager Admin Console, go to
              Hosts > All Hosts.Reselect the hosts you selected in Step 2.Select Actions for Selected > Remove
              from Cloudera Manager.
