Step 6: Install Parcels
If you selected parcels for the installation method, the Install
      Parcels page reports the installation progress of the parcels you selected
    earlier. 
After the parcels are downloaded, progress bars appear representing each cluster host. You
      can click on an individual progress bar for details about that host.
After the installation is complete, click Continue.
The Inspect Cluster page displays.
