Removing a Host From a Cluster
Removing a host from a cluster leaves the host managed by
    Cloudera Manager and preserves the Cloudera Management Service roles (such
    as the Events Server, Host Monitor,and so on).

Minimum Required Role:
Full
            Administrator. This feature is not available when using Cloudera
            Manager to manage Data Hub clusters.


In the Cloudera Manager Admin Console, click the Hosts tab.Select the hosts to delete.Select Actions for Selected > Remove
              From Cluster. The Remove Hosts From
            Cluster dialog box displays.Leave the selections to decommission roles and skip removing the Cloudera
          Management Service roles. Click Confirm to proceed with removing
          the selected hosts.
