Renewing a License
You can upload a license file to renew a CDP Private Cloud Base license.

Minimum Required Role:
Cluster
            Administrator (also provided by Full
          Administrator) This feature is not available when using Cloudera
            Manager to manage Data Hub clusters.


Download the license file and save it locally.In Cloudera Manager, go to the Home
          page.Select
              Administration > License.Click Update License.Browse to the license file you downloaded.Click Upload.
