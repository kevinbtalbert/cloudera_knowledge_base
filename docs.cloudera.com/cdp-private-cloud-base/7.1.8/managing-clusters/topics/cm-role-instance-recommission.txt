Recommissioning Role Instances

Minimum Required Role:
Operator
        (also provided by Configurator,
          Cluster Administrator, Limited
          Cluster Administrator , Full
          Administrator) 
Click the service that contains the role instance you want to
          recommission.Click the Instances tab.Check the checkboxes next to the decommissioned role instances
          to recommission.Select Actions for
              Selected > Recommission,
          and then click Recommission to start the process. A
          Recommission Command pop-up displays that shows each step or
          recommission command as it is run. When the process is complete, a
             is added in front of Recommission Command.Restart the role instance.
