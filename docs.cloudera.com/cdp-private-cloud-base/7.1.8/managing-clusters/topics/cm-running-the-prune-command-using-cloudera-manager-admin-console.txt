Running the Prune Command Using Cloudera Manager Admin
        Console


Minimum Required Role:
Cluster
            Administrator (also provided by Full
          Administrator) This feature is not available when using Cloudera
            Manager to manage Data Hub clusters.

To prune the S3Guard metadata in the DynamoDB table using the
        Cloudera Manager Admin Console: Go to Administration > AWS Credentials.Locate the credential associated with the S3 data and click Actions >  Run S3 Guard Prune Command.
