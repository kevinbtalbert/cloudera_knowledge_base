Starting All the Roles on a Host
You can start all the roles on a host from the Hosts
    page.

Minimum Required Role:
Operator
        (also provided by Configurator,
          Cluster Administrator, Limited
          Cluster Administrator , Full
          Administrator) 

Click the Hosts tab.Select one or more hosts on which to start all
        roles.Select Actions for
              Selected > Start Roles on
            Hosts.
