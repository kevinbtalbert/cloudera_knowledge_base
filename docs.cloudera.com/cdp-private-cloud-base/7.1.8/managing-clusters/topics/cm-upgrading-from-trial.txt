Upgrading from a CDP Private Cloud Base Trial to CDP Private Cloud Base
You can upgrade your license from a trial license to a CDP Private Cloud Base
    license.

Minimum Required Role:
Cluster
            Administrator (also provided by Full
          Administrator) This feature is not available when using Cloudera
            Manager to manage Data Hub clusters.


Purchase a CDP Private Cloud Base Edition license from Cloudera.On the License page, click Update License.Click the Select License File field.Browse to the location of your license file, click the file, and click
          Open.Click Upload.
