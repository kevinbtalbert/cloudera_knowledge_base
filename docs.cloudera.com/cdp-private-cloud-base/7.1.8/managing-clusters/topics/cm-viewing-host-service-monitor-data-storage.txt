Viewing Host and Service Monitor Data Storage
The Cloudera Management Service page shows the current disk space consumed and its
    rate of growth, categorized by the type of data stored. For example, you can compare the space
    consumed by raw metric data to daily summaries of that data.

Select Clusters > Cloudera Management
              Service.Click the Charts Library tab.
