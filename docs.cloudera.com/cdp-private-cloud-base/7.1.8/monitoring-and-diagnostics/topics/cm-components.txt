Components
The Components page lists every component installed on this host.
    This may include components that have been installed but have not been added as a service (such
    as YARN, Flume, or Impala).
This
      includes the following information:


Component - The name of the component.

Version - The version of Cloudera Runtime from which each
        component came.

Component Version - The detailed version number for each
        component.
