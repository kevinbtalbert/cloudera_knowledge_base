Configuring Log Directories
You can configure log directories for a cluster or a service.

Minimum Required Role:
Configurator (also provided by
          Cluster Administrator,
Limited Cluster Administrator , and
          Full Administrator)


Do one of the following:
Cluster:

On the Home > Status tab, click a cluster name.
Select Configuration > Log Directories.
Edit a 
role_name Log
                Directory property.

Service:

Go to a service.
Click the Configuration tab.
Select 
role_name
                  (Service-Wide) > Logs.
Edit the Log
                Directory property.

Enter a Reason for change, and then click Save
            Changes to commit the changes.Click the Cloudera Manager logo to return to the Home page.Click the 
          icon that is next to any stale services to invoke the cluster restart wizard.
