Configuring Time-Series Query Results
A time-series query returns one or more time series or scalar values. By default a
    maximum of 250 time series will be returned.

Minimum Required Role:
Configurator (also provided by
          Cluster Administrator,
Limited Cluster Administrator , and
          Full Administrator)

To change this value, select Administration > Settings.In the Advanced category, set the Maximum Number Of Time-Series Streams
            Returned Per Time-Series Query or the Maximum Number of Time-Series Streams
            Returned Per Heatmap property.Click Save Changes.
