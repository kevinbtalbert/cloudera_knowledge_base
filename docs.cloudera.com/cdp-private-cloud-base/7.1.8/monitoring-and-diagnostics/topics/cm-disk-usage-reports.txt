Disk Usage Reports 
There are two types of disk usage reports: Current Disk Usage By
      Directory and Historical Disk Usage By
    Directory.
To use these reports, select one or more directories to watch by clicking the
             icon for the directory.Alternatively, you can select multiple directories, and then click Actions for Selected > Include selected directories in disk usage reports.
