Logs and Events
For information about problems, check the logs and events.

 Logs present log information for services, filtered by role, host,
        or keywords as well log level (severity).
 The topic Viewing the Cloudera Manager Server Log
        contains information on the server and host agents.
The Events tab lets you search for and display events and alerts
        that have occurred within a selected time range filtered by service, hosts, or
        keywords.
