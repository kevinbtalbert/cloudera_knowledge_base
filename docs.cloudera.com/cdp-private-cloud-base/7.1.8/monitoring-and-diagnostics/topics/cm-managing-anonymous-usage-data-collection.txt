Managing Anonymous Usage Data Collection
Cloudera Manager sends anonymous usage information using Google Analytics to Cloudera.
    The information helps Cloudera improve Cloudera Manager. By default, anonymous usage data
    collection is enabled. You can disable this
    property.Select Administration > Settings.Under the Other category, set the Allow Usage Data Collection
          property.Enter a Reason for change, and then click Save Changes to
        commit the changes.
