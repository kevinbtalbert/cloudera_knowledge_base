Managing Dashboards
You can create, clone, edit, export, import, and remove dashboards.
To manage
      dashboards, select Charts > Manage Dashboards. 

Create Dashboard - create a new dashboard.

Clone - clones an existing dashboard.

Edit - edit an existing dashboard.

Export - exports the specifications for the dashboard as
          a JSON file.

Import Dashboard - reads an exported JSON file and
          recreates the dashboard.

Remove - deletes the dashboard.
