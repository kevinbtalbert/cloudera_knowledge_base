Modifying the Health Threshold
You can configure the threshold for when a health alert is raised.

Minimum Required Role:
Limited Cluster
            Administrator  (also provided by Full
          Administrator and Cluster
          Administrator)


Select
            Administration > Alerts.Click  to the right
          of Health Alert Threshold.
Select Scope > Event
            Server.Click the Main category.Select the Bad or Concerning
          option.Enter a Reason for change, and then click Save
            Changes to commit the changes.Click the Cloudera Manager logo to return to the Home page.Click the 
          icon that is next to any stale services to invoke the cluster restart wizard.
