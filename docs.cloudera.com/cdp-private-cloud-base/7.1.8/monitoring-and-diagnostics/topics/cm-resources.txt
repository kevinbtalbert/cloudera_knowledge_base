Resources
The Resources page provides information about the resources
    (Memory, disk, and ports) used by every service and role instance running on the selected
    host.
Each entry on this page lists:
The service name
The name of the particular instance of this service
A brief description of the resource
The amount of the resource being consumed or the settings for the resource


The resource information provided depends on the type of resource:

Memory - The number of bytes consumed

Disk - The disk location where this service stores
          information

Ports - The port number being used by the service to
          establish network connections
