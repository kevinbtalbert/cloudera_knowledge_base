Saving Charts to a New Dashboard
You can save new or existing charts to a new dashboard.

Optionally, modify the chart properties.If the chart was created with the Chart Builder, optionally type a name for the
          chart in the Title field.Do one of the following:
New chart: Click Save.
Existing chart: Move the cursor over the chart, and click the  icon at the top right.

Optionally, edit the chart name.Select the Add chart to a new custom
            dashboard option.Enter a dashboard name.Click Save Chart. The new
          dashboard appears on the menu under the top-level Charts
          tab.
