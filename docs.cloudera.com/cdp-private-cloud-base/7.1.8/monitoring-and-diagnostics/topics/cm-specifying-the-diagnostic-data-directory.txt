Specifying the Diagnostic Data Directory
You can configure the directory where collected diagnostic data is
    stored.

Select
              Administration > Settings.Under the Support category, set the Diagnostic Data Bundle Directory
          to a directory on the host running Cloudera Manager Server. The directory must exist and
          be enabled for writing by the user cloudera-scm. If this field is left
          blank, the data is stored in /tmp.Enter a reason for the change and click Save
            Changes.
