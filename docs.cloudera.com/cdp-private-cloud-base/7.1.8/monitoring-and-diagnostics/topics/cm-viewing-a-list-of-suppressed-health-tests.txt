Viewing a List of Suppressed Health Tests
You can view a list of suppressed health tests from the
      Configuration menu.

Minimum Required Role:
Configurator (also provided by
          Cluster Administrator,
Limited Cluster Administrator , and
          Full Administrator)


From the Home page or the
            Status page of a cluster,
          select Configuration > Suppressed Health and Configuration
              Issues. 
Select Status > Non-default.A list of suppressed health tests and configuration
          issues displays.To limit the list to health tests, enter “health test” in the Search box.
