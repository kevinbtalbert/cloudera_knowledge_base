Viewing and Debugging Spark Applications Using Logs
You can view overview information about all running Spark applications.
Go to the YARN Applications page in the Cloudera Manager Admin
          Console. To debug Spark applications running on YARN, view the logs for the NodeManager
          role. Open the log event viewer. Filter the event stream.For any event, click View Log File to view the entire log
          file.
