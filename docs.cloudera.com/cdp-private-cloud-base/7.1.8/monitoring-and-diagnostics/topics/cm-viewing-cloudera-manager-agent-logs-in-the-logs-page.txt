Viewing Cloudera Manager Agent Logs in the Logs Page
You can view the Cloudera Manager Agent logs in the Logs
    page.

Select
            Diagnostics > Logs
          on the top navigation bar.Click Select Sources to display the log source
        list.Uncheck the All Sources checkbox.Click  to the left of Cloudera Manager and select the
            Agent checkbox.Click Search.
