Viewing Cloudera Manager Server Logs in the Logs Page
You can view Cloudera Manager Server logs in the Logs
    page.

Select
            Diagnostics > Logs
          on the top navigation bar.Next to Sources, select the Cloudera Manager
            Server checkbox and deselect the other options.Adjust the search criteria and click Search.
