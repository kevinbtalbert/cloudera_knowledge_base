Viewing Host Details
You can view detailed information about each host, such as name, IP address, and rack
    ID, and more from the All Hosts page.
To view detailed host information, click
              Hosts > All
            Hosts.Click the name of one of the hosts. The Status page is displayed for the host you
          selected.Click tabs to access specific categories of information. Each tab provides various
          categories of information about the host, its services, components, and
          configuration.
