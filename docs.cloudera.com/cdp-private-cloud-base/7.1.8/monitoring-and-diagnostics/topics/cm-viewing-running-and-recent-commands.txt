Viewing Running and Recent Commands
You can view and run recent commands for a cluster, service, or role. You can also
    view the details of recent or running commands.
Viewing Running and Recent Commands For a ClusterYou can use the command indicator to view the running and recent commands for a     cluster.Viewing Running and Recent Commands for a Service or RoleYou can view running and recent commands for a service or role instance in the       Commands tab.Command DetailsYou can view command details. The details available for a command depend on whether     the command is running or recently completed.Parent topic: Monitoring Services
