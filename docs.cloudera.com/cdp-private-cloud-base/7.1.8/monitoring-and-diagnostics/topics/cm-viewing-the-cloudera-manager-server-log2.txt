Viewing the Cloudera Manager Server Log
You can access the Cloudera Manager Server Log through the
      Diagnostics menu or the server host.

Select Diagnostics > Server
            Log on the top navigation bar.Optionally, you can view the Cloudera Manager Server log at
            /var/log/cloudera-scm-server/cloudera-scm-server.log on the Server
          host.
