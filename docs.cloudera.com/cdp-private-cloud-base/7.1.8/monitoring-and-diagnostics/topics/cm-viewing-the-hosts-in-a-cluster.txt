Viewing the Hosts in a Cluster
You can view the hosts in a cluster. The All Hosts page displays
    a list of the hosts filtered by the cluster name.

Do one of the following:
Select Clusters > Cluster
                  name > Hosts.
In the Home screen, click  in a full-form cluster table.
