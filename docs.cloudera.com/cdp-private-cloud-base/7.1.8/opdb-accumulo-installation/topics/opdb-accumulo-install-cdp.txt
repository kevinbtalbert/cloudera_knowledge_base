Install CDP
Before you install the Operational Database powered by Apache Accumulo (OpDB) parcel,
  you have to install CDP Private Cloud Base and add the mandatory
  services.
The OpDB service is only supported on CDP Private Cloud Base. For full
   cluster installation Cloudera recommends following the CDP Private Cloud Base Installation Guide.
For a FIPS compliant OpDB service you have to follow the FIPS deployment CDP setup steps. For
   more information, see Installing and Configuring CDP with FIPS.
Ensure you install  the following mandatory services for OpDB powered by Apache Accumulo:

HDFS


ZooKeeper
