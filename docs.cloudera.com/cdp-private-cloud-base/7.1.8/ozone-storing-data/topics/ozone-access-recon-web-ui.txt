Access the Recon web user interface
You can launch the Recon web user interface from Cloudera Manager. Recon starts its
        HTTP server over port 9888 by default. The default port is 9889 when auto-TLS is enabled. 

Go to the Ozone service.

Click Recon Web UI.




The Recon web user interface loads in a new browser window.
