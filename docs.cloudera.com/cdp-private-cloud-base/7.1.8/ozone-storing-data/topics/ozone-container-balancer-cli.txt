Container balancer CLI commands
You can run the following commands in the cluster.


To start the service, run the following command ozone admin
                        containerbalancer start
To stop the service, run the following command ozone admin
                        containerbalancer stop
To check the status of the service, run the following command ozone
                        admin containerbalancer status
