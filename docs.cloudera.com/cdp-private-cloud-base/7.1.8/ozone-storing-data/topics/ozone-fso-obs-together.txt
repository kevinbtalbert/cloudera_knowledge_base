Files and Objects together
Bucket Layout concept is now introduced in Ozone that helps you with the unified design
    representing files, directories, and objects stored in a single system.
A single unified design represents files, directories, and objects stored in a single system.
      Ozone performs this by introducing the bucket layout concept in the metadata namespace server.
      With this, a single Ozone cluster with the capabilities of both Hadoop Compatible File System
      (HCFS) and Object Store (like Amazon S3) features by storing files, directories, objects and
      buckets efficiently. Also, the same data can be accessed using various protocols.
