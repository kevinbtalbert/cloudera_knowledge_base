Enabling S3 Multi-Tenancy 
You must perform the following steps to enable the S3 multi-tenancy
        feature.

Log in to Cloudera Manager UI

Navigate to Clusters

Select the Ozone service

Go to Configurations

Search for Enable Ozone S3 Multi-Tenancy and select the
                    checkbox

Click Save Changes

Restart the Ozone service
