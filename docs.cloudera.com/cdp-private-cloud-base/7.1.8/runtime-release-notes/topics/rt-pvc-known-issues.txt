Known issues in Cloudera Runtime 7.1.8
You must be aware of the known issues and limitations, the areas of impact, and
        workaround in Cloudera Runtime 7.1.8.noteCDSW does not support RPM-based installation on CDP Private Base. (RPM installation is
        deprecated and only supported on HDP and CDH 5. For CDH6 and onward, Cloudera recommends you to
        use CSD-based installations.)
