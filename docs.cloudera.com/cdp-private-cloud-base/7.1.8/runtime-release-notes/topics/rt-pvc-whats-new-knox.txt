What's New in Apache Knox
This topic lists new features for Apache Knox in Cloudera Runtime 7.1.8.
Certified OpenIDConnect (OIDC) support for Knox SSO
Users can now enable authentication via OpenIDConnect (OIDC). OIDC is an industry standard
        for using a third party as an identity provider for web applications.
