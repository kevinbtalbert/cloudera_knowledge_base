Add the HttpFS role
You can use Cloudera Manager to add the HttpFS role.

Go to the HDFS service.
Click the Instances tab.
Click Add Role Instances.
Click the text box below the HttpFS field. The Select
                Hosts dialog box displays.
Select the host on which to run the role and click
                OK.
Click Continue.
Check the checkbox next to the HttpFS role and select Actions for Selected > Start.
