Limitations of erasure coding
The limitations of erasure coding include non-support of XOR codecs and certain HDFS
        functions.
EC does not support:
XOR codecs
certain HDFS functions such as hflush, hsync, concat, setReplication, and
                    truncate. For more information, see Erasure Coding Limitations.
