JWT algorithms
Similarly to Kafka, Schema Registry also uses Jose4J for validating the JWT tokens and
  their signatures.

This library supports a range of signing algorithms: HS256, HS384, HS512,
     RS256, RS384, and RS512.For more information, see Bitbucket jose4j Wiki.
