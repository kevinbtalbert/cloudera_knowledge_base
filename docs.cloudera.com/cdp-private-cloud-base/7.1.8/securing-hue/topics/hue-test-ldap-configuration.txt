Testing the LDAP configuration
You can test your Hue LDAP configuration without restarting the Hue service. Add the
      values and save your changes.

Configure Hue to authenticate with LDAP by using search bind or direct bind.

Add a user and group name for Test LDAP
              Configuration.

Click Save Changes.

Select
              Actions > Test
                LDAP Configuration.

Click Test LDAP Configuration:






Click Restart Hue. When the test succeeds, 
              log in to the Hue Web UI.
