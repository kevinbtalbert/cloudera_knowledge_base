Ranger UI authorization
Reference information on Ranger UI authorization, when configuring Ranger AD
      integration.
To configure the users, groups, and roles that can access the Ranger portal or
                        its services, select Settings > Users/Groups/Roles in the top
                        menu.



A user can be a User, Admin, or Auditor:



Only users with the Admin role can edit Ranger policies.
