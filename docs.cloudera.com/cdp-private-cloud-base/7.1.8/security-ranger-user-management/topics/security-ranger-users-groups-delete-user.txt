Deleting a user
How to delete a user in Ranger.
Only users with the "admin" role can delete a user.Select Settings > Users/Groups.
The Users/Groups page appears. 


Select the check box of the user you want to delete, then click the Delete icon
                  () at the right of the User List menu
               bar.






Click OK on the confirmation pop-up.
