SRM Command Line Tools
Overview of the command line tools shipped with SRM.
importantSRM ships with the srm-control,
        srm-driver, srm-service, and srm command
      line tools. Use the srm-control tool to the manage replication of topics and
      consumer groups. Do not use the srm-driver, srm-service, and
        srm tools, use Cloudera Manager instead to manage the SRM driver and
      service.
