Manage reports
As a database administrator, you can view which columns and tables are used for joins and make changes to the data layout to optimize the performance of the query.
noteThe view reports feature is not available in the DAS-Lite version.

Viewing the Read and Write reportThe Read and Write report displays a list of tables belonging to the selected database     along with the read count, write count, and other details for each table.Viewing the Join reportThe Join report provides you the details of joins between all the tables in a     database.
