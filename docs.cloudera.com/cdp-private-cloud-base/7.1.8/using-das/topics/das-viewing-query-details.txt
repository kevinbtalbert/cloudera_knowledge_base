Viewing the query details
The Query Details tab provides information such as, the Hive query
    ID of the query, the user who executed the query, the start time, the end time, the total time
    taken to run the query, the tables that were read and written, the application ID, the DAG
    ID(s), the session ID, the thread ID, and the queue against which the query was run.
Click the Edit button in the Qury Details
            section to edit and rerun the query.
