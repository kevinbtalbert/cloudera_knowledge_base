External tables based on a non-default schema
If you define a schema for external tables, you need to know how to create the table using the hive.sql.schema 
        table property to prevent problems with client connections to the tables. Examples for widely-used databases show you how
        to create such tables.
        
    
 The handling of schema differs from DBMS to DBMS. In this task, you follow
            examples to create external tables that the following databases can understand. 
MariaDB
MS SQL
Oracle
PostgreSQL
