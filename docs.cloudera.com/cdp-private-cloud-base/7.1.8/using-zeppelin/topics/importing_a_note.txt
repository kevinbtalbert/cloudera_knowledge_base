Import a Note
Use the following steps to import an Apache Zeppelin note.
To import a note from a URL or from a JSON file in your
         local file system:
Click "Import note" on the Zeppelin home page:


Zeppelin displays an import dialog box:


To upload the file or specify the URL, click the associated box.
By default, the name of the imported note is the same as the original note. You
               can rename it by providing a new name in the "Import AS" field.
