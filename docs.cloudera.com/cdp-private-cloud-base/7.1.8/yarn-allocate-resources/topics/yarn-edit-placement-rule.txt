Editing placement rules
The Queue Manager UI enables you to edit previously created placement
    rules.

In Cloudera Manager, select YARN Queue Manager UI.
A graphical queue hierarchy is displayed in the Overview
          tab.

Go to the Placement Rules tab.
A list of placement rules is displayed.

In the Action column click on the Edit
            rule icon in the row of the placement rule you want to edit.
The Edit Placement Rule dialog box is
          displayed.

Edit the placement rule.

Click Save.
