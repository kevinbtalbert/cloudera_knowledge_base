Configuring YARN Queue Manager dependency
 If you add the YARN Queue Manager service to the cluster after
    installing the cluster, you must configure the YARN Queue Manager
    dependency.

In Cloudera Manager, select the YARN service.
Click Configuration.
Search for Queue Manager service.
Select the YARN Queue Manager checkbox.


Click Save Changes.
Restart the YARN and the YARN Queue Manager
        services.
