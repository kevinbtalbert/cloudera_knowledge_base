Disabling YARN Ranger authorization support
You can disable any YARN Ranger repository through the Ranger Admin UI.

In Cloudera Manager, selec the Ranger service.

Click Ranger Admin Web UI.
Takes you to the Ranger Admin Web UI site.

Log in to the Ranger Admin Web UI.

Find the applicable YARN Ranger repository.

Click the edit icon.

Find the Active Status field and select
                        Disabled.

Click Save.
