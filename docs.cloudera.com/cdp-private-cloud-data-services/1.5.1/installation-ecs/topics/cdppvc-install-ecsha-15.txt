Checking Nodes and Pods in the UI
You must check the nodes and pods in the UI.

Log in to Cloudera Manager.
Navigate to the ECS service.
Click the Web UI drop-down.
Click ECS Web UI. The Kubernetes web UI page
    opens in a new tab.
Check the Nodes and Pods on the Web UI.
