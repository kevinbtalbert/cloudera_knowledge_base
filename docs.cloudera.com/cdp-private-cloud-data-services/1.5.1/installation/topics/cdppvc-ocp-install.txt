Installation on the OpenShift Container Platform (OCP)



CDP Private Cloud Data Services pre-installation checklistBefore starting the installation, you must ensure that you have configured all the         required hardware and software. There are several pre-installation tasks that you must         complete using Cloudera Manager and OpenShift Container Platform. Installing in internet environmentFollow the steps in this topic to install CDP Private Cloud.Installing in air gap environmentYou can launch the Private Cloud installation wizard from Cloudera Manager and follow         the steps to install CDP Private Cloud Data Services in an air gap         environment where your Cloudera Manager instance or your Kubernetes cluster does not have         access to the Internet.Uninstall CDP Private Cloud Data ServicesYou can uninstall CDP Private Cloud Data Services from your CDP Private Cloud Base Cloudera Manager.
