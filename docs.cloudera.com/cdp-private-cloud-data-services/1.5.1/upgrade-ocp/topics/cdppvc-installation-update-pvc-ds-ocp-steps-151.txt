OCP upgrade steps for CDP Private Cloud Data Services
        1.5.1
Updating CDP Private Cloud Data Services 1.4.1 or 1.5.0 to 1.5.1
        on OCP requires you to upgrade earlier OCP versions to a version supported by CDP Private Cloud Data Services 1.5.1. This process involves incremental
        OCP version upgrades in conjunction with CDP Private Cloud Data Services
        updates and validations. 
OCP upgrade steps for CDP Private Cloud Data Services 1.4.1
                to 1.5.1
Starting point: CDP Private Cloud Data Services 1.4.1 on OCP
                4.8.



Upgrade OCP from 4.8 to 4.9.


Upgrade OCP from 4.9 to 4.10.


Update CDP Private Cloud Data Services from
                            1.4.1 to 1.5.1.


Perform post-upgrade validations of existing resources.


Upgrade OCP from 4.10 to 4.11.


Perform validations on all data services to verify that the new
                            and preexisting data and workloads continue to function properly.
