Migrate Ranger Policies
How to migrate Ranger policies for Streaming clusters from HDF to CDP Private Cloud Base.
Use the Ranger Console (Ranger UI) or the Ranger REST API to import or export policies. For
      more information, see the Ranger documentation for Runtime or the Ranger
        documentation for HDP
