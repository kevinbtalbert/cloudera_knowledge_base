Getting the JDBC
      driver
You learn how to download the Cloudera Hive and Impala JDBC drivers to give clients
      outside the cluster access to your SQL engines.

Download the lastest Hive JDBC driver for CDP from the
                  Hive JDBC driver download page. 

Go to the Impala JDBC driver page, and download the
               latest Impala JDBC driver.

Follow JDBC driver installation instructions on the
               download page.
