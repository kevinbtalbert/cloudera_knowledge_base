Add Balancer Role to HDFS
Use HDFS Balancer for balancing the data. This section is not applicable if you are
        upgrading to CDP Private Cloud Base 7.1.7.
The HDFS Balancer is a tool for balancing the data across the storage devices of a HDFS
            cluster. For more information on HDFS Balancer, see Balancing data across an HDFS
                cluster.
