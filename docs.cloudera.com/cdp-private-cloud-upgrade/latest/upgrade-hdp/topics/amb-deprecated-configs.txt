Remove Hive Ranger property
This section is not applicable if you are upgrading to CDP Private Cloud Base 7.1.7. 
From the Hive service, navigate to the Advanced Configuration Snippet (Safety
                Valve) option and you must remove the
                org.apache.ranger.authorization.hive.authorizer.RangerHiveAuthorizerFactory
            property.
