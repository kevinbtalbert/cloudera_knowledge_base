HBase
Post-migration, Atlas hook for HBase is not enabled by default on AM2CM migrated
        clusters. you must manually enable Atlas hook.
To enable the Hive Metastore hook in your Cloudera Manager instance:
Log in to Cloudera Manager
Select the HBase service
Select Configurations tab
Search for Enable Atlas Hook that the HBase service
                    depends on and check this.
Select Save Changes
