Enabling SAC manually on Spark
Post-migration to CDP Private Cloud Base, you must manually enable SAC on Spark and
        Hive.

Log in to your Cloudera Manager instance
Navigate to Clusters
Select the Spark service.
Select Configuration tab
Search for Atlas Service and select the
                    atlas check box
Click Save Changes
Navigate to Clusters
Select the Hive service
Select Configuration tab
Search for Atlas Service and select the
                    atlas check box
Click Save Changes
Restart  the affected Services.
