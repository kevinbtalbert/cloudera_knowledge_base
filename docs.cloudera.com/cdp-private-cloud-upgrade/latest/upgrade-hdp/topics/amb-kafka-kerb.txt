Kafka cluster Kerberos
If the CDP cluster is kerberised, then you must set this parameter. This setting is
        applicable only if you are upgrading from HDP 2.6.5 to CDP Private Cloud Base
        7.1.6.

Log into Cloudera Manager UI

Navigate to Clusters

Select the Kafka service

Go to Configurations

Search for kerberos.auth.enable and enable the
                    parameter
