Kerberos principal
YARN service and Kafka service in Cloudera Manager does not take non-default
        principal names. This section is not applicable if you are upgrading to CDP Private Cloud
        Base 7.1.7.
Currently, the YARN service in Cloudera Manager does not take non-default principal
            names. The YARN principle name is not migrated to Cloudera Manager if you use any custom
            principal name in the YARN service on Ambari. 
Currently, the Kafka service in Cloudera Manager does not take non-default principal
            names. The Kafka principle name is not migrated to Cloudera Manager if you use any
            custom principal name in the Kafka service on Ambari.
