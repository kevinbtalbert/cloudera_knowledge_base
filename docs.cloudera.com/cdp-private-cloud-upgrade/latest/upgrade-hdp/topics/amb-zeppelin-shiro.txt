Zeppelin Shiro configurations
If you had Zeppelin Shiro configurations in the HDP cluster, then you must configure
        them manually on the CDP Private Cloud Base cluster. 
Perform the following steps:

Log into Cloudera Manager

Navigate to Clusters

Select the Zeppelin service

Click the Configuration tab

Search for Shiro and add the configurations

Click Save Changes
