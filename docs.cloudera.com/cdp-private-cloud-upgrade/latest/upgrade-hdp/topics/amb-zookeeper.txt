ZooKeeper
You must complete the following steps to start the ZooKeeper service. 
Remove or move the myid file from the ZooKeeper server hosts. The path
          is ${dataDir}/myid. For example, 
# mv /hadoop/zookeeper/myid
            /hadoop/zookeeper/myid.bak


Start ZooKeeper service from Cloudera Manager
