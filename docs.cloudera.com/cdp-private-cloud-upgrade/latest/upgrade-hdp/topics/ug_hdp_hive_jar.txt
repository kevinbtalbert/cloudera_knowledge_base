Download the Pre-Upgrade Tool JAR for Compaction
You must download the software that you run later to detect tables for compaction and
        to perform that compaction. 

SSH into the host running the Hive Metastore.

Change to the /tmp directory.

Download the pre-upgrade tool JAR.
$ wget https://repo.hortonworks.com/repository/releases/org/apache/hive/hive-pre-upgrade/3.1.0.3.1.0.258-3/hive-pre-upgrade-3.1.0.3.1.0.258-3.jar
