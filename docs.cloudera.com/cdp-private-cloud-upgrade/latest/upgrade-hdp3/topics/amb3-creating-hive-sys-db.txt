Create HIVE sys database
You need to perform this procedure when you are upgrading to Cloudera Manager to 7.7.1
    version and runtime to 7.1.8 version.


Log in to Cloudera Manager
Navigate to Clusters
Select the HIVE service
Click Actions drop-down.
Click Create Hive Sys database
