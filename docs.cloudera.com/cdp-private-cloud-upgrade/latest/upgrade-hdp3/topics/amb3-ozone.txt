Ozone
To start Ozone service after migrating to the CDP Private Cloud Base cluster with
        ranger plugin enabled. Perform the following steps:


Log in to Cloudera Manager
Navigate to Clusters
Select the Ozone service
Select Configurations tab
Search for Ozone Manager Environment Advanced
                    Configuration Snippet (Safety Valve)
Add Key - PARCEL_DIRNAMES and Value - CDH
