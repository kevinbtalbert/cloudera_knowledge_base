Upgrading Ambari
You must understand the considerations and the process to upgrade from the current
        version of Ambari to Ambari 7.1.x.x.

It creates a copy of the current Ambari         database and meta information.Setting up a local repositoryYou need to set up a local repository, update the version repository base URLs, and edit the repository configuration file.​Preparing to Upgrade AmbariYou must prepare to upgrade Ambari to 7.1.x.x.
