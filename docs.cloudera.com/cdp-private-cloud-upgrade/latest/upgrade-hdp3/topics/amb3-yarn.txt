Perform the following post migration steps.  
Start job historyYou must start job history after migration.Yarn Mapreduce framework jarsGPU SchedulingYou must enable GPU on the CDP cluster if GPU was enabled on the HDP         cluster.YARN CGroupsIf YARN CGroups was enabled in the HDP cluster, you must manually enable the         Host-level CGroups settings.Reset ZNode ACLsIf you have a kerberized cluster, you manually have to reset the ZNode ACLs after         upgrading from HDP to CDP.Placement rules evaluation engineIf you are upgrading to Cloudera Runtime 7.1.6 or higher you need to learn about the new   placement rules evaluation engine.Setting the owner and permissions of /user/yarn If you want to add a VPC or Compute cluster after migration, ensure that the correct         owner and permissions are set for /user/yarn in HDFS.
