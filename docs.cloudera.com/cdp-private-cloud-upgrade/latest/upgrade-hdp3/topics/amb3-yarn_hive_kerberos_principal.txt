Kerberos principal
Kafka service in Cloudera Manager does not take non-default principal names. This
        section is not applicable if you are upgrading to CDP Private Cloud Base 7.1.8.
Currently, the Kafka service in Cloudera Manager does not take non-default principal
            names. The Kafka principal name is not migrated to Cloudera Manager if you use any
            custom principal name in the Kafka service on Ambari.
