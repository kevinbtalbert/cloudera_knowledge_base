Base
Runtime
Cloudera Runtime in CDP Private Cloud Base sustains several dozen open-source database and big data projects—Apache HDFS, Apache Hive, Apache HBase, Apache Impala, and similar projects—for your data workloads.

Cloudera Manager for CDP Private Cloud Base
Cloudera Manager helps manage, configure, and monitor CDP Private Cloud Base clusters.
  

Flow Management
Flow Management is a no-code data ingestion and management solution powered by Apache NiFi.

Streaming Analytics
Streaming Analytics is a framework for real-time stream processing and streaming analytics powered by Apache Flink.
