CDP Private Cloud Base 7.1.4 Release Summary
October 2020
Cloudera is pleased to announce the release of Cloudera Data Platform
(CDP) Private Cloud (PvC) Base version 7.1.4. CDP PvC Base 7.1.4 is a
cumulative maintenance release carrying forward all features and
functions from previous versions 7.1.1, 7.1.2 and 7.1.3, plus the
following additional platform support and features:
Platform Support

Oracle 12 database support for in-place upgrade from HDP 2.x to
DC-Ambari
Oracle 19 database support for upgrade & fresh installs
SLES 12 SP5 operating system support for fresh installs and
upgrades
Postgres 11 database support for fresh installs & DC-Ambari

Features

Enable “create table” legacy behavior for Hive with purge table
by default, and keyword “managed” by turning on legacy
configuration to help customer upgrade and migrate
Absolute mode option in Yarn Queue Manager can be used to
configure actual vCores and memory, in addition to relative
quantities
Enable multi-tenancy in Apache Atlas with a new security
features that restricts viewing entity details and displays
classifications only created/authorized by current tenant
Tech preview of ACL sync between HDFS and Apache Ranger
using the new ranger RMS service is now available
