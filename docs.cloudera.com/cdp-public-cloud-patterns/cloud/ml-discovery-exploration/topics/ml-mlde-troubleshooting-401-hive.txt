Troubleshooting: 401 Unauthorized when accessing Hive 
Problem: Session returns a 401: Unauthorized error when you are accessing a Hive
  database.
Check that the Hive data warehouse is working.
Solution: Follow these steps to check that Hive data warehouse is running, and restart if
    needed.
In the control plane, go to Data Warehouses.
Click Virtual Warehouses, and select the data warehouse for the
     connection.
Check the status of that data warehouse, and make sure that it is running or in a good
     state.
