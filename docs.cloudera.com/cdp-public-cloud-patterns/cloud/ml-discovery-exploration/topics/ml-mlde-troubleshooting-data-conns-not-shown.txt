Troubleshooting: Some connections not shown
The data application doesn't show all data connections.
See the error message to fix the workload password. To set the workload password, follow the
   procedure in Setting the workload password. You will need to restart the data
   application after setting the workload password.
Alternatively, you may need to click Sync under the Data Connections to get all of the data
   connections available at the project level.
