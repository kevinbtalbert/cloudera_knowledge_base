About sample data
For this pattern, we use a dataset about airlines and flights. This dataset is available
  in the Virtual Warehouse when you create a non-default Database Catalog with the Load Demo Data
  option enabled.
Airlines data (available in CDP)
The airline data consists of the following four tables:
airlines
airports
