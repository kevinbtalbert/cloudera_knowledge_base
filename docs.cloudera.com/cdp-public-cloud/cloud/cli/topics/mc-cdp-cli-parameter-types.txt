CDP CLI parameter types
This section describes the format of common CLI parameter types.
Timestamp
Format the timestamp parameter according to the ISO 8601 standard.
For example:


YYYY-MM-DDThh:mm:ss.sssTZD (UTC)
YYYY-MM-DDThh:mm:ss.sssTZD (with offset)
YYYY-MM-DD
Unix time in seconds
