Accessing CLI help
CDP CLI includes help that can be accessed by using the cdp help
  command. 
To get more information about a certain CDP CLI,
   use:cdp <module-name> <command-name> helpFor example:

cdp iam add-ssh-public-key help

cdp iam help|grep group
You can also find all of the CDP CLI commands in the CDP
        CLI Reference documentation.
