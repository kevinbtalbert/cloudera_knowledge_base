Quickstarts


These CDP Quickstarts walk you through the prerequisite set up required
in your cloud provider account, as well as creating a CDP credential
and registering an environment in CDP.



AWS Onboarding Quickstart
Learn step-by-step how to connect CDP to your AWS account, so that you can begin to provision clusters and workloads.

Azure Onboarding Quickstart
Learn step-by-step how to connect CDP to your Azure subscription, so that you can begin to provision clusters and workloads.
 

GCP Onboarding Quickstart
Learn step-by-step how to connect CDP to your GCP project, so that you can begin to provision clusters and workloads.
