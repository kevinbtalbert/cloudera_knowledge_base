CDP CIDR
CDP CIDR includes the following IP ranges:

Control Plane Region
IP Ranges

us-west-1
35.80.24.128/27, 35.166.86.177/32, 52.36.110.208/32, 52.40.165.49/32

eu-1
3.65.246.128/27

ap-1
3.26.127.64/27

When creating your own security groups for CDP, you must open required ports to all of these IP
   ranges.
