SSH key pair
When registering an environment, you will be asked to upload an SSH key pair. The minimum SSH key size is 2048 bits.
The SSH key will be used for root-level access to Data Lake and Data Hub instances. 
If you need to generate an SSH key, you can use typical SSH key generation steps, such as
      those described in Quick steps: Create and use an SSH public-private key pair
        for Linux VMs in Azure .
