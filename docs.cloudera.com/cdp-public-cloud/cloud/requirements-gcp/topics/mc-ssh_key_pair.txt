SSH key pair 
When registering an environment, you will be asked to provide an SSH key pair for admin
  access to CDP. The minimum SSH key size is 2048 bits.
You will need to paste the public SSH key in CDP during environment registration. If you need
   help generating an SSH key, refer to https://www.ssh.com/ssh/keygen/.
