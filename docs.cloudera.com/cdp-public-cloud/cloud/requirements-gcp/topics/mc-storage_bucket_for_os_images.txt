Storage bucket for OS images 
By default, CDP creates a GCS bucket for storing OS images used for Data Lake and Data
  Hub VMs, but you can optionally pre-create it if your organization requires it. 


It must have a name using the following convention:
     <CDPtenantID>-<ProjectID> 


You can your CDP tenant ID as described in Obtain CDP tenant ID.
