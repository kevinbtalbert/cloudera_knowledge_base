CDP Reference Architectures


CDP Reference Architecture documents help you understand, plan, and
deploy solutions using CDP services and components. Older documents are
also available.



CDP Private Cloud Base
Provides high-level design and best-practices for deploying CDP Private Cloud Base in data centers.

Disaster Recovery
Discusses generic recommendations and best practices that will help you plan and understand disaster recovery solutions for Cloudera Data Platform.
 

Security
Provides a detailed security reference architecture for Cloudera Private Cloud Base. The architecture reflects the four pillars of security engineering best practice: perimeter, data, access, and visibility. These layers of security are applied to ensure the confidentiality, integrity, and availability of data to meet the most robust of regulatory requirements.
