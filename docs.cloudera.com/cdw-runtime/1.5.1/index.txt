CDP Private Cloud Data Services Release Notes
CDP Private Cloud Data Services Release Summary


Data Warehouse Runtime provides tightly integrated Apache Hive, Apache
Impala, and Hue services for the CDP Private Cloud Data Warehouse Data
Service. The Data Warehouse release
notes include
information about Data Warehouse Runtime.
Information here describes how to use Apache Hive and Apache Impala SQL
from clients that connect to your Virtual Warehouse in CDP Private
Cloud. Also covered is the Hue interactive query editor, which you can
open from a Virtual Warehouse to run SQL queries.
