Datagen connector
The Datagen connector can be used as a source to generate random data. The Datagen
  connector works out of the box, no mandatory field is required to use the connector. The Datagen
  connector is useful when you want to experiment and try out SQL Stream Builder or to test your SQL
  queries using random data.

Template: datagen
You do not need to provide any type of information when using the Datagen connector and
     template.
