Other supported statements
Beside the supported DDL, DML and SELECT, the supported statements also include
  DESCRIBE, SHOW and SET that you can use in SQL Stream Builder.
The following table summarizes the supported statements for in SQL Stream Builder:

Operation
Description



DESCRIBE TABLES
Describing the schema of a table


SHOW TABLES
Showing a list of existing tables, views, functions, databases or
        catalogs


SHOW VIEWS


SHOW FUNCTIONS


SHOW DATABASES


SHOW CATALOGS


SET
Setting properties of session

For more information about Flink SQL, see the Apache Flink documentation.
