Install Sentry
Sentry is an RPC server that stores authorization metadata in an underlying relational
  database and provides RPC interfaces to retrieve and manipulate privileges. It supports secure
  access to services using Kerberos.
Before you begin:
If you are using CDH 6.3.0 or later, you must install HDFS.
If you are using CDH 5.13.0 or later, you must install either HDFS or Isilon.
