Obtain the Kafka service name
You will need the Kafka service name when you are configuring SMM. It is helpful to
        obtain it in advance.

In Cloudera Manager, go to your Kafka service.

Select any available chart, and select Open in Chart
                        Builder from the configuration icon drop-down.

Find $SERVICENAME= near the top of the display. 

This value is the Kafka service name that you can use for
            SMM configuration.
