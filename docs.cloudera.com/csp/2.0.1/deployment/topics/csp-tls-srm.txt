Enable TLS/SSL for SRM
Learn how to Enable TLS/SSL for SRM.

In Cloudera Manager select Streams Replication Manager. Go to Configuration.
Find and enable the Enable TLS/SSL for Streams Replication Manager
            Driver property.

Find and enable the Enable TLS/SSL for Streams Replication Manager
            Service property.

noteIf a third-party truststore is used, you have to manually configure and add
            truststore information via Cloudera Manager.
