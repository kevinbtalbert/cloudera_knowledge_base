Monitoring Cluster Replications by Quick Ranges


You can filter the cluster replications by time range. You can choose any of the
                following filter values from the drop-down to display the cluster replication
                details in the selected time range:
Last 1 Hour
Last 6 Hours
Last 24 Hours
Last 2 Days
