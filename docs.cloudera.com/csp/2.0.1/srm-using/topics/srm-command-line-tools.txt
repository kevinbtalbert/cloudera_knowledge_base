SRM Command Line Tools
Overview of the command line tools shipped with SRM.
importantSRM in CSP includes the srm-control,
        srm-driver, srm-service, and srm.sh
      command line tools. Use the srm-control tool to the manage replication of
      topics and consumer groups. Do not use the srm-driver,
        srm-service, and srm.sh tools, instead use Cloudera
      Manager to manage the SRM driver and service.
