Deleting a schema
You can use the Schema Registry REST API to delete schemas.

You can access the Schema Registry API Swagger documentation directly from the UI. To
                do this, append your URL with: /swagger/
For example: http://localhost:9090/swagger/


Find the DELETE /api/v1/schemaregistry/schemas/{schema_name}
                    API under "Schema".

Replace {schema_name} with the schema you wish to
                    delete.

Invoke the API.
