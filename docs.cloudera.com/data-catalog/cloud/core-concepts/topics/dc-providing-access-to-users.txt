Providing role access
You must provide the required role access to use Data
        Catalog.

From Cloudera Data Platform > Management Console >
                        Environments > Select an Environment > select Actions
                    drop-down > Manage Access.

Search for the user who requires Data Catalog access > Select the check-box,
                    either EnvironmentAdmin or EnvironmentUser > Click Update
                        Roles.











Navigate back to the Environment details page and select
                        Actions > select Synchronize Users to FreeIPA
Allow the sync operation to complete and the changes to take effect.

noteFor tenant specific roles and related permissions, see
                            Account roles.
