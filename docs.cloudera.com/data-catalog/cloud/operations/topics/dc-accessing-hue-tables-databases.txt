Accessing Hue tables and databases
When you log into Data Catalog, you can access web links to the Hue instances from
        the Data Catalog Search page or the Asset Details page.
You can access the Hue tables and databases by using one of the following options in
            Data Catalog,

Log into Cloudera Data Catalog UI.

When you navigate to the entity search page, you can click on the  icon present at
                    the end of the row which opens the Link to Experiences
                    dialog.

Directly on the selected asset's Asset Details page, you can click
                        Link to Experiences.
