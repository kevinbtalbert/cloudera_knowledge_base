Adding Lookup Files
When you have too many allowed and denied entries and cannot add them inline, you can
        create allowed or denied files with one value in each line and add them to your DSL. 

Click Resources in the right panel on the New
                        Custom Profiler Rules page.

Click + icon on the Lookups tab. The New
                        Lookup File page appears.

Enter the name of the new Lookup file.

Click Choose File to upload the file.

Click Save.
