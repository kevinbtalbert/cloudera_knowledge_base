View Data Asset Schema
From the Asset Details Schema page, you can view the schema of the data asset for structured data (such as Hive
      tables) from the relevant metadata repositories (such as Atlas).

From the Data Catalog search page, select an asset.
The Asset Overview window opens.

Click Schema. 
The Schema table shows the data asset schema as
               retrieved from Apache Atlas. 

(Optional) To edit tags:


Click Edit Tags.


Click the (+) icon.


Select or deselect the tags you choose, then click
                        Save.

You can now manage and edit tags at the table level.
