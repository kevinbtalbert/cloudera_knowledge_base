This is the first release of the Data Catalog service. 

   Data Catalog is a service within Cloudera Data Platform that enables you to understand, manage, secure, and govern data assets across enterprise data lakes. Data Catalog helps you understand data across multiple clusters and across multiple environments (on-premises, cloud, and IOT).
  
Data Catalog enables data stewards across the enterprise to work with data assets in the following ways:

Organize and curate data globally
Understand where relevant data is located
Understand how data is interpreted for use
Understand how data is created and modified
Understand how data access is secured and protected, and audit use
