This release of the Data Catalog service introduces the following new
  features.
 Data Catalog is a service within Cloudera Data Platform that enables you to understand,
   manage, secure, and govern data assets across enterprise data lakes. Data Catalog helps you
   understand data across multiple clusters and across multiple environments. 
Data Catalog introduces the following new features:

Ability to run Sensitivity Profiler jobs
Ability to collaborate with other users - you can now rate assections, add comments about asset collections and respond to the comments of other users.
