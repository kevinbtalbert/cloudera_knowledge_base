This release of the Data Catalog service introduces the following new
        features.
 Data Catalog is a service within Cloudera Data Platform that enables you to understand,
            manage, secure, and govern data assets across enterprise data lakes. Data Catalog helps
            you understand data across multiple clusters and across multiple environments. 
Data Catalog introduces the following changes and bug fixes:

UI improvements for search capabilities
Ability to search Hive and HBase assets using Column tag
