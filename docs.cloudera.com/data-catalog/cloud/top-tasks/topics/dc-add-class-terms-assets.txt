Adding Classifications / Terms for selected assets
You can add classification or terms that can be associated with an asset.

From Data Catalog > navigate to the search page.

You can perform one or more of the following:


Select Add Classifications / Terms on action button in search
                            page.


Select Add Classifcations / Terms in classification widget on
                                Asset Details page.



On the Add Classifications / terms slider, click on the
                        Add icon against classification / term.

Enter other values in the fields, if required and click
                        Save.
