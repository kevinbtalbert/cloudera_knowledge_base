Deleting a workload secret for Cloudera Data Engineering Spark Jobs using CLI
You can delete an existing secret for Cloudera Data Engineering (CDE) Spark Jobs using
    CLI.
./cde credential delete --name <workload-credential-name> 
For example: ./cde credential delete --name workload-cred-1
