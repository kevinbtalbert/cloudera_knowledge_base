Creating a Session using the CDE CLI [Technical Preview]
The cde session create command allows you to create a new
    Session.
Ensure that you are using a version of CDE 1.19 or higher for your Virtual Cluster and
      ensure that you have downloaded the CLI client. For more information, see   Using the Cloudera Data
        Engineering command line interface.

Run the following command in the CDE CLI: 

cde session create --name <session-name> --type <pyspark/spark-scala>
