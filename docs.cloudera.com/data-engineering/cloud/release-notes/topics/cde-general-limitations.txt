Limitations
Cloudera Data Engineering (CDE) has the following limitations.

No support for Data Lake resizing
CDE does not support Data Lake resizing.
Running raw Scala code in Cloudera Data Engineering 


When setting the Log Level from the user interface, the setting is not applied to the raw
       Scala jobs.
Do not use package <something> in the raw Scala job file as Raw
       Scala File is used for Scripting and not for Jar development and packaging.
