Network and availability
The "Network and availability" options allow you to customize the networking settings of
  your cluster.
On the Network and Availability page, provide the following to specify the networking
   resources that will be used for your cluster:



Parameter
Description



Select Subnet
If your environment includes a single subnet, that subnet is preselected and it cannot
        be changed. If your environment includes more than one subnet, you can select the subnet(s)
        in which your cluster will be provisioned. If you would like to deploy your
         Data Hub in multiple availability zones, you should select multiple subnets.
