Create a cluster 
You can quickly create clusters from default cluster definitions within an existing
    environment.
To create a cluster:

Navigate to the Management Console > Environments > click on an
    environment where you would like to create a cluster > click Create Data Hub
    Cluster.
Provide the following:
Select Cluster Definition.
In the Services section, select a specific cluster definition, for example Data
            Engineering for AWS. 
Under General Settings > Cluster Name, provide some name for your cluster.

Click on Provision Cluster to trigger cluster creation.
