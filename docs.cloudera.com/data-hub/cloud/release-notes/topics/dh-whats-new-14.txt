October 2021Cloudera Docs
October 2021
This release of the Data Hub service introduces new features.
Encrypting EBS volumes with a CMK generated from AWS CloudHSM (October 4)
New documentation with steps for encrypting Data Hub's EBS volumes with a CMK generated from
    AWS CloudHSM is  available. See Encrypting EBS volumes with a CMK generated from AWS
     CloudHSM.

Data Hub maintenance upgrades (October 26)
You can perform a maintenance (or "hotfix") upgrade on certain Data Hub clusters. For more
        information see Upgrading Data Hubs.
