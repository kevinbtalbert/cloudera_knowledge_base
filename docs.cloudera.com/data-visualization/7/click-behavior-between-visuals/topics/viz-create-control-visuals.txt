Creating control visual
In Cloudera Data Visualization, you can have one (source) visual act as a filter that
    controls the behavior of the target visual, inside the same application.

Open (or create) a visual based on US State Population Over Time dataset.

The specifics for a sample visual follow:

Dataset:
US State Population Over Time
Visual type: map
Geo shelf:
state field
Measures shelf:
population field, as sum(population)
Filters shelf:
year field, as year in (2010)



Click Refresh Visual.






Name it as Control Visual and save it.
