Creating the dashboard

The following steps use a calendar heat map visual, such as the one described in
        Calendar heat maps, which is built on the dataset SFPD Incidents. 

Start a new dashboard by clicking New Dashboard.

In the Dashboard Designer, open SFPD Incidents -
            Calendar Heat Map, built according to the instructions in Calendar heat
            maps. Ensure that you are using the SFPD Incidents dataset.
The visual appears in the dashboard.

To edit the visual, click the Pencil icon in the top right
          corner of the visual.

Under Dimensions, select pddistrict and add it
          to the Filters shelf. Repeat the action with
            category and date.






Name and save the dashboard.
