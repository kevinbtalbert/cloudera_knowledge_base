Preparing Excel files for data import
Learn how you can import data from Microsoft Excel or other similar formats into
    Cloudera Data Visualization.
You must save the files in character-delimited format.

In Microsoft Excel, open the file that contains the data you are importing.

Click Save as, then select Comma Separated Values (.csv).
        

Choose a file name, such as MyCSVdata, and click
          Save.
