Data connections in Cloudera Data Visualization
Cloudera Data Visualization allows you to create connections to many types of external
    data sources. You can set up connections to the different data sources that you can then use to
    build visuals.
Cloudera Data Visualization supports the following connection types:

Hive
Impala
MariaDB
MySQL
PostgreSQL
Spark SQL
SQLite (not supported in Cloudera Data Warehouse (CDW))
Snowflake (Technical Preview)

For more information, see Connecting to data.
