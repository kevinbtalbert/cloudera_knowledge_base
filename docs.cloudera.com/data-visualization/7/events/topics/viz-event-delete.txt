Deleting events


This articles demonstrates how to delete an event defined on a dataset. Let's delete the
        event All application events. 


On the main navigation bar, click Data.

Select the Campaign dataset. Dataset side navigation appears, open at
            Dataset Detail view.

In the side navigation menu, click Events.

In the Events list, find the event to delete, and click
            Delete.

In this example, we are deleting the event All application events.





In the confirmation window modal, click Delete.






The event End no longer appears in the Events
          interface.
