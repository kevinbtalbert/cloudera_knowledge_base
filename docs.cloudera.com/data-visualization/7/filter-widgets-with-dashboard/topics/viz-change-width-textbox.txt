Changing the width of the filter text box
This option allows you to change the width of the filter text box in a dashboard
    filter.

Open an existing or new dashboard in Edit mode and add a new filter

Hover over the filter you added until the controls appear on the top right.

Click the Gear icon.
The Settings modal window appears.

Click Display Settings and enter a value in the Width
            (in filter mode) box to change the width of the filter box.

Click APPLY.
