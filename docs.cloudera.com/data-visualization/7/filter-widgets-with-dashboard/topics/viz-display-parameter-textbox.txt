Displaying a textbox parameter
This option allows you to specify a parameter filter text box in a dashboard filter. 

If checked, this option supersedes all other values included in the Specified
          values option, on the Values tab of the
          Settings modal window. Instead, you can enter a parameter value you
        want to use at runtime, or search for data that matches certain values.. It creates a simple
        search box filter.


Open an existing or new dashboard in Edit mode and add a new filter

Hover over the filter you added until the controls appear on the top right.

Click the Gear icon.
The Settings modal window appears.

Check the Display a text box parameter option on the
            Display Settings tab.

Click APPLY.
A parameter filter textbox appears.

Enter a value in the textbox.
In this example All-Bran is added.



The visual displays only All-Bran details.
