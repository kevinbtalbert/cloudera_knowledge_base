Using single value, with dropdown menu
You can use this option to select a single value from a dropdown menu in a dashboard
    filter.

Open a dashboard in Edit mode and hover over a filter that is already added to
          the dashboard until the controls appear on the top right.

Click the Gear icon.
The Settings modal window appears.

Click the Display Settings tab, then check the Allow
            only one item to be selected at a time option and the Select
            values from a dropdown menu sub-option.

Click APPLY.
