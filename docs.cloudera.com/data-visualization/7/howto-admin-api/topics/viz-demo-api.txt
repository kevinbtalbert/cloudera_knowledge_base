Demo admin API
This demo page can be used to view how network requests are formed for accessing the
            Cloudera Data Visualization admin APIs.

Open Site Settings

Scroll to Advanced Settings at the bottom of the
                    page.

Add the following:

ADMIN_API_DEMO_LIST = ['visuals', 'datasets', 'connections', 'users', 'groups', 'roles', 'segments', 'workspaces', 'filterassociations']                    


Click Save


Once enabled, the a page will be available on
                    [domain]/arc/apps/apidemo.
