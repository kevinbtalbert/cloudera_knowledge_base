Adding dashboards to tabs
Each tab in a CDP Data Visualization application contains one or more
    dashboards.
Add a dashboard to the first tab in your application by clicking a dashboard in the
            Dashboards menu on the right side. 
You can preview the information about the dashboard by hovering over it.
noteYou can select recently viewed dashboards, or the dashboards in the selected
            dataset.

The newly added dashboard appears in the tab container.
Click Save.
