Changing datasets in applications
In a Cloudera Data Visualization application, you can switch between datasets and use
    dashboards from different datasets, even when sourced from different connections.

In the Dashboards menu, select a different dataset.
In this example, the World Life Expectancy dataset is used.



Select a dashboard available on the dataset you used. 
In this example,  the Life Expectancy Dashboard is selected.

The newly added dashboard appears in the tab.
