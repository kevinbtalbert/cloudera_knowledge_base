Changing dashboard widthCloudera DocsChanging dashboard width

You can change the width of a dashboard in CDP Data
      Visualization.
To change the width of a dashboard, navigate to
      the General menu, and adjust the value of the Dashboard
        width option. The default value is 1024. To have dashboards extend to the full
      width of the page, clear the field.
