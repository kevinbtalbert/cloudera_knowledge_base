Basics styles

You can adjust several style options for basics in CDP Data
      Visualization dashboards. To get to the options in the Dashboard Basics
      menu, follow these steps: 

On the right side of Dashboard Designer, click the
            Style menu.

In the Style menu, click Dashboard
          Basics.

Select one of the options.
