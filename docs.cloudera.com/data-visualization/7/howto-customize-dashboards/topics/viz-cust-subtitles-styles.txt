Dashboard subtitles styles

You can adjust several style options for dashboard subtitles
      in CDP Data Visualization dashboards. To get to the options in the Dashboard
        Subtitle menu, follow these steps:

On the right side of Dashboard Designer, click the
            Style menu.

In the Style menu, click Dashboard
            Subtitle.

Select one of the options.
