Visual basics styles in dashboards


You can find several basic style options for dashboard visuals in CDP Data
        Visualization.
To get to the options in the Visual Basics menu, follow these
        steps:


On the right side of Visual Designer, click the
            Style menu.

In the Style menu, click Visual
          Basics.

Select one of the options.
