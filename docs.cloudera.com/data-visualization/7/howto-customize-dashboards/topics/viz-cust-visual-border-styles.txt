Visual border styles in dashboards


You can find several border style options for dashboard visuals in CDP Data
        Visualization.
To get to the options in the Visual Borders menu, follow these
        steps:


On the right side of Dashboard Designer, click the
            Style menu.

In the Style menu, click Visual
          Borders.

Select one of the options.
