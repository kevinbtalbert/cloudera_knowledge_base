Visual subtitles styles in dashboard


You can adjust several style options for visual subtitles in CDP Data Visualization
        dashboards.
To get to the options in the Visual Subtitle menu, follow these
        steps: 


On the right side of Dashboard Designer, click the
            Style menu.

In the Style menu, click Visual
          Subtitle.

Select one of the options.
