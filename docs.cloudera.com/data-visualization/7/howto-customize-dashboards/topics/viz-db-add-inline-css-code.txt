Adding inline CSS code to dashboards
You can apply simple formatting to visuals with CSS code snippets in CDP Data
    Visualization dashboards.
To add simple formatting to all the visuals in
      the dashboard, navigate to the dashboard Custom Style menu, and enter a
      CSS code snippet under the Enter in-line CSS option. You can select the
        Autocomplete option, too.To apply this operation at the level of a
        specific visual, see Adding inline CSS code to visuals.
