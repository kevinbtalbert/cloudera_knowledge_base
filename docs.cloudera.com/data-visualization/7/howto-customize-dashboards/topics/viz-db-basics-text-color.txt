Dashboard text color
You can change the color of the text in the dashboard in CDP Data
    Visualization.
To change the color of the text in the
      dashboard, navigate to the dashboard Style menu, select the Dashboard
        Basics menu, and select a new color in the Text Color
        selector.You can choose from one of the standard system colors, specify a hexadecimal or
        RGB color, or select one in the continuous color palette. You can also adjust color opacity.
        The default is no color.The following image shows the dashboard text before
