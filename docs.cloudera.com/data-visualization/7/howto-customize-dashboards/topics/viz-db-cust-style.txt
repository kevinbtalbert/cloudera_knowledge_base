Customizing dashboard style
Cloudera Data Visualization supports several options for styling on dashboards.To
    change the options for dashboard style, follow these steps. 
 On the right side of Dashboard Designer, click the
            Style menu.

In the Style menu, you can see a list of options that contain
          all the style adjustments.





Select the option you need.
