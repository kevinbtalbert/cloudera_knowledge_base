Resetting dashboard style
You can reset the styling of a dashboard to its default settings.To change the style
    back, follow these steps. 
Open the dashboard in Edit mode.

 On the right side of the Dashboard Designer, click the
            Style menu.

In the Style menu, click  at the top
          right corner to reset the dashboard to its original style.






Save the dashboard to preserve the new style.
