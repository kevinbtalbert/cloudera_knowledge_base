Visual legend font size in dashboards

You can change the font size of a legend in
      visuals in CDP Data Visualization dashboards. To change the font size used of the legend in
      all visuals that appear on this dashboard, navigate to the dashboard
        Style menu, select the Visual Axes and Labels
      sub-menu, and make adjustments in the Legend Font Size
      selector.
