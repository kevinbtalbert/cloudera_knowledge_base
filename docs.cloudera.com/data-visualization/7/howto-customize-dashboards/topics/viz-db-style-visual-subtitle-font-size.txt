Visual subtitle font size in dashboards

You can change the font size of the visual
      subtitle in CDP Data Visualization dashboards. To change the font size of the text in
      subtitles of all visuals of the dashboard, navigate to the dashboard
        Style menu, select the Visual Subtitle menu,
      and then adjust the Font Size selector.
