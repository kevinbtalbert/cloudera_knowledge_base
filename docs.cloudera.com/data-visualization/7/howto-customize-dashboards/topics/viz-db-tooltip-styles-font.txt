Tooltip font family in dashboards

You can change the font of the tooltips in CDP
      Data Visualization dashboards. To change the font of the tooltips in all visuals of the
      dashboard, navigate to the dashboard Style menu, select the
        Tooltip sub-menu, and then select a new font in the Font
        Family selector.
