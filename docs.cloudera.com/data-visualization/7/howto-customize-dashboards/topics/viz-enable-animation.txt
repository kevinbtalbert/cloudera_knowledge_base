Enabling animations


Cloudrea Data Visualization enables visual transitions in dashboards for the following
        visual types: bars, lines, combo, area, scatter, grouped bar, histogram, sparklines, pie,
        packed bubbles, and correlation heat map.
noteThis feature is only available when you enable visual transitions at site level. For
        more information, see Enabling visual transitions.


To enable animations on specific dashboard visuals, follow these steps:


Open the dashboard.

In Dashboard Designer, select the
            Settings menu.

Click the General menu to expand it.

Select the Enable animations option.
