Enabling email for a dashboard

You can email dashboards in Cloudera Data
      Visualization.
noteBefore using the
        email function, you must enable the feature in site settings. Only site administrators can
        customize email settings to allow users to share visuals through email. For instructions,
        see Email settings.To allow the user to email dashboards, navigate to
      the Downloads menu and select the Enable emails
      option.
