Using filter labels inline

You can change the position of the dashboard
      filter labels in CDP Data Visualization. To show labels inline with selectors, navigate to the
      dashboard Style menu, select the Filters menu,
      and select the Display labels inline option. By default, the filter
      labels appearover its selection controls.noteThis option is only available when the
        dashboard has filters across the top.
