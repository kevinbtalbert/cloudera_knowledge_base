Selecting page size for PDF downloads

You can select the page size when downloading a PDF in
      Cloudera Data Visualization.
To change the page size of PDF downloads,
      navigate to the Downloads menu, and select the appropriate option in
      the PDF Page Size menu.Data Visualization supports the Letter
        and A4 standard sizes.
