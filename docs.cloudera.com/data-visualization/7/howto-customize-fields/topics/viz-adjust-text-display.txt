Adjusting text display options
Data Visualization enables you to adjust the text display options for field values
    before rendering visuals.
To use text functions, follow these steps:Click the field you plan to modify, to open the Field
            Properties menu.Under Field Properties, click to expand the Text
            Functions menu.



From the Text Functions menu, select one of the
          options.

Uppercase converts text to upper case; returns the value of
                  upper([field])
Lowercase converts text to lower case; returns the value of
                  lower([field]).

A check mark precedes the selected aggregate function.
Click Refresh Visual to see the changes.
To the text mask from a field, click on the selected mask. It removes the check
            mark.
