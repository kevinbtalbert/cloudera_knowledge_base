Customizing display format
Data Visualization has robust support for formatting the display of individual
    values.
To change the display format of a field, follow these
      steps:Click the field you plan to modify, to open the Field
            Properties menu.Under Field Properties, click Display
            Format.


Change the display format to another category with specialized options:
By default, the Category for display format is
              None.
For a complete list of supported formats, see Display Format
            Reference.
