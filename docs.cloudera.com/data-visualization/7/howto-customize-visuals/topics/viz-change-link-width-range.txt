Changing link width rangeCloudera DocsChanging link width range


To change the width of the link, navigate to the Links menu, and
        change the minimum and the maximum in the Link width range option.
        The default setting is 4-10 pixels.



Compare the appearance of a visual with link width range of 2-2 (on the
        left) with a visual that uses a link width range of 4-4 (on the right).
