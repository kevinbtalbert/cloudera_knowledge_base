Changing the map style of a visual
CDP Data Visualization provides six different interactive map style
    options.

On the right side of Visual Designer, click
            Settings.
In the Settings menu, click Map
          Style.

To change the map style of an interactive map visual, select one of the options in
          the Map Style dropdown menu. The options are Default,
            Subtle, Dark, Light, Muted, and Satellite.
