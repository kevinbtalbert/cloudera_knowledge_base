Changing node frictionCloudera DocsChanging node friction


On the right side of Visual Designer, click
            Settings.
In the Settings menu, click
          Nodes.

To change the friction of a node, change the value of the Node
            friction option.The default setting is 0.3, and the valid range is 0 to
            1.

Compare the appearance of a visuals with node friction of 0.09,
              0.5, and 0.9.
