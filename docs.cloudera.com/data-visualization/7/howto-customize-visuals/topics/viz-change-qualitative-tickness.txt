Changing qualitative thicknessCloudera DocsChanging qualitative thickness


This task applies to Gauge charts.
By default, the width of the qualitative (outer) arc of the visual is set to 10 pixels. To
        change it, navigate to the Marks menu, and enter the new value for
        the Qualitative Thickness option.



Note how adjusting this setting changes the appearance of the visual.
