Changing source node mark


On the right side of Visual Designer, click
            Settings.
In the Settings menu, click
          Nodes.

To change the mark of the source node, change the selection in the
            Source Node Mark menu.
The options are:

circle (default)
square
diamond
cross
triangle-up
triangle-down




Compare the appearance of a visual with source node circle mark, and source node
              square mark.
