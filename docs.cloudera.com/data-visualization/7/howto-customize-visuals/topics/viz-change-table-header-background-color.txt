Changing table header background color

You can change the background color of a table
      header.
On the right side of Visual Designer, click the
            Settings menu.


In the Settings menu, click Table
            Colors.


To change the backround color of the header in a table visual, make changes in the
            Header Background selector. You can select one of the palette
          colors, or specify a custom color.
