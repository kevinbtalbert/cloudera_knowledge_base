Changing table odd rows background color

You can change the background color of the odd rows in a table
      visuals.
On the right side of Visual Designer, click the
            Settings menu.


In the Settings menu, click Table
            Colors.


To change the background color of the odd rows in a table visual, make changes in
          the Odd Rows Background selector. You can select one of the palette
          colors, or specify a custom color.
