Changing table border color

You can change the color of the cell borders in a table
      visual.
On the right side of Visual Designer, click the
            Settings menu.


In the Settings menu, click Table
            Colors.


To change the color of the cell borders in a table visual, make changes in the
            Border selector. You can select one of the palette colors, or
          specify a custom color.
