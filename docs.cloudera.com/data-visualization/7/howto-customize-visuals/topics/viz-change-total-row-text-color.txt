Changing totals row text color

You can change the text color of the totals row in a table
      visual.
On the right side of Visual Designer, click the
            Settings menu.


In the Settings menu, click Table
            Colors.


To change the text color of the totals row in a table visual, make changes in the
            Totals Row Text selector. You can select one of the palette
          colors, or specify a custom color.
