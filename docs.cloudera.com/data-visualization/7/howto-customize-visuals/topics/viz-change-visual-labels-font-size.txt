Label font size in a visual

On the right side of Visual Designer, click the
            Settings menu.


In the Settings menu, click Visual
            Title.

To change the font size used by the labels of a visual, navigate to the
            Customizing Visual Axes and Labels menu, and make adjustments in
          the Label Font Size selector.
