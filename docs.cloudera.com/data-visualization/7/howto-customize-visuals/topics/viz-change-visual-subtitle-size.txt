Changing font size in the subtitle of a visual 

On the right side of Visual Designer, click the
            Settings menu.


In the Settings menu, click Visual
            Subtitle.


To change the font size used by the subtitle of a visual, navigate to the
            Visual Subtitle menu, and make adjustments in the Font
            Size selector.
