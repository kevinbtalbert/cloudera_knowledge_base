Trellis font size in a visual

On the right side of Visual Designer, click the
            Settings menu.


In the Settings menu, click Visual
            Title.

To change the font size used of a trellis in a visual, navigate to the
            Customizing Visual Axes and Labels menu, and make adjustments in
          the Trellis Font Size selector.
