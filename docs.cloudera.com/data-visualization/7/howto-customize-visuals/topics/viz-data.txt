Customizing data for visuals


Follow these steps to get to the data options:


On the right side of Visual Designer, click
            Settings.

In the Settings menu, click Data.




 Under Data, select one of the following options:

Ordering NULL rowsChanging number of rows to downloadChanging maximum number of rows to fetchDisabling incremental queriesDisplaying NULL rowsEnabling URL linksSpecifying qualitative range as percentagesParent topic: Customizing visual settings
