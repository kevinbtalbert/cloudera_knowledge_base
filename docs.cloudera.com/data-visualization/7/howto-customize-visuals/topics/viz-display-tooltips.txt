Displaying tooltips


CDP Data Visualization allows you to enable tooltips in Sparklines visuals.
This setting is available on all visuals.

To turn the Tooltips option on or off, navigate
      to the Marks menu, and select the Display
        Tooltips option.

On some visuals, such as sparklines, turning on the Display
        tooltips optionautomatically selects all of the following options for showing
      values in the tooltip. To show less than all four, de-select that option.
