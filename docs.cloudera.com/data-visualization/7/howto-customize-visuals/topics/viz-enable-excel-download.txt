Enabling Excel download in visualsCloudera DocsEnabling Excel download in visuals
You can enable or disable Excel data download for each visual in an XLS
    file.
This setting is available on all visual types. noteThe site
        administrator can override this setting by defining Enable "Download as
          CSV/Excel" in the Site Settings interface. For more
        information, see Downloading maximum number of rows in CSV
      files.

On the right side of Dashboard Designer, click Settings > Downloads from the VISUAL menu.






Select Enable "Download as CSV/Excel".




To disable Excel download, de-select this option.
