Enabling first value in tooltips
In Sparklines visuals, CDP Data Visualization  enables you to
    view the first value in a tooltip.
This setting is only available on Sparklines
        Visuals.

To display the first value in a tooltip, navigate to the Marks menu,
        select Display Tooltips, and then select the Enable first
          value in tooltips option.



If this option is selected, notice that the First value appears in
        the tooltip.
