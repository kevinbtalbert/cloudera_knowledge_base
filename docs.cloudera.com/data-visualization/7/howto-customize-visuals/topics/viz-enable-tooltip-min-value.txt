Enabling minimum value in tooltips
In Sparklines visuals, CDP Data Visualization enables you to
    view the minimum value in a tooltip.
This setting is only available on Sparklines
        Visuals.
To display the minimum value in a tooltip,
      navigate to the Marks menu, select Display
        Tooltips, and then select the Enable minimum value in
        tooltips options.

If this option is selected, notice that the Minimum value
        appears in the tooltip.
