Customizing map serverCDP Data Visualization provides two digital map services: Google Maps and     Mapbox.Specifying mapbox key for a visualYou can override the Mapbox token in the Site Settings interface     by specifying a Mapbox token at the visual level.Specifying atlas mapbox server for a visualYou can specify an Atlas Mapbox server at the visual level. With Atlas Mapbox server,     you can run Mapbox APIs on your own infrastructure, whether on your own private cloud or     on-premises.Parent topic: Customizing visual settings
