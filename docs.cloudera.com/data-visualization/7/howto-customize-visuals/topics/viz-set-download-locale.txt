Setting the locale for CSV downloadsCloudera DocsSetting the locale for CSV downloads
By default, the locale settings for CSV download format match the system settings of
    the Internet browser. However, it is possible to specify a different locale for
    download.
This setting is available on all visual types. 

On the right side of Dashboard Designer, click Settings > Downloads from the VISUAL menu.






Set the language in the  Set "Download as CSV" Locale  drop-down
          menu.






Here, the default locale is en-US, and we use the Set "Download as CSV" Locale menu to
      select an alternate locale,Germany.
