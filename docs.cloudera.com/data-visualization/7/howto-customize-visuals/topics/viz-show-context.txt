Showing the context menu

You can click on a part of a visual and see the context menu.
      This option is only available for Table and Queries visuals. 
To turn the context menu option on or off,
      navigate to the Marks menu, and select Show detail data
        button in context menu. The following image shows the context menu in a table
        visual after the option is selected. To hide the context menu in a visual, de-select this
        option.

The context menu includes the following options:
Show Detailed Data 
Include: Filter Data Using the Include Option in the Context Menu
exclude: Filter Data Using the Exclude Option in the Context Menu
