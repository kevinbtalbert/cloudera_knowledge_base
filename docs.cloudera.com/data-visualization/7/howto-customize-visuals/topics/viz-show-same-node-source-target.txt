Showing the same source and target values

This task applies to Network visuals.

On the right side of Visual Designer, click
            Settings.
In the Settings menu, click
          Nodes.

To ensure that the end points are defined by the same source and destination
          values, select the Source & target node data come from the same base
            values option.
