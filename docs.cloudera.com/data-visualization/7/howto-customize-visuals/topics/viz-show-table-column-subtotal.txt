Showing column page totals
In CDP Data Visualization, you can add Table, and Queries visuals report values in
        rows by default. But there is an option of adding a subtotals row to a table and use it to
        present column page totals, which is the total of the currently viewed page.

On the right side of Visual Designer, click
                        Settings in the VISUAL section
                    of the menu.






Open the Data Summary menu and select the
                        Show page totals option.







The visual now has a row showing the total of currently viewed page:
