Showing header column borders
In CDP Data Visualization, header columns in Table and Queries visuals have a border by
    default.

To show or hide header column borders for Tables and Queries:

On the right side of the Visual Designer, click
            Style in the VISUAL section of the menu
            bar.


In the Style menu, click Table Style.


Select or deselect the Display header column borders option.

        this option.
