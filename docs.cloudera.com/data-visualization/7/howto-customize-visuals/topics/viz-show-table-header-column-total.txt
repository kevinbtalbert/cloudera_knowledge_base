Showing header for column totals
In CDP Data Visualization, if you use column totals in your Table and Queries visuals,
    you can also add a header for column totals.

On the right side of Visual Designer, click
            Settings in the VISUAL section of the
          menu.






Open the Data Summary menu and select the Show totals
            header option.




noteThis option is only applied to the visual if Show
              totals is selected.
