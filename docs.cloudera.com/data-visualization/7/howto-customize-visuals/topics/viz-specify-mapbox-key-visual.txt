Specifying mapbox key for a visualCloudera DocsSpecifying mapbox key for a visual
You can override the Mapbox token in the Site Settings interface
    by specifying a Mapbox token at the visual level.

On the right side of Visual Designer, click
            Settings.
In the Settings menu, click Map
            Server.

Enter a mapbox key for the visual in the Map key for visual text
            box.This setting will override the Mapbox Token setting in
          the Site Settings menu.importantYour organization must obtain its own
            credentials for Google Maps and MapBox. See Site
                Settings > API Keys for Interactive Map
              Visual.
