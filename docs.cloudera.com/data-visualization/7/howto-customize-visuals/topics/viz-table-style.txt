Customizing table style
In CDP Data Visualization, you can define various style options for your table
    visuals.
Changing table templatesIn CDP Data Visualization, you can choose from various table templates when creating a     Table or a Queries visual.Changing table header alignmentIn CDP Data Visualization, you can table header alignment for Tables and     Queries.Adjusting table widthYou can set a Table, a Cross Tabulation or a Queries visual to full width to ensure     that they fill the entire width of the page.Wrapping column contentsIn CDP Data Visualization, you can wrap content in table columns. If the text does not     wrap, you can hover over the text to view full text.Showing column totalsIn CDP Data Visualization, you can add Table, Cross Tabulation, and Queries visuals     report values in rows by default. But there is an option of adding a totals row to a table and     use it to present column totals.Showing header for column totalsIn CDP Data Visualization, if you use column totals in your Table and Queries visuals,     you can also add a header for column totals.Showing column page totalsIn CDP Data Visualization, you can add Table, and Queries visuals report values in         rows by default. But there is an option of adding a subtotals row to a table and use it to         present column page totals, which is the total of the currently viewed page.Showing header for column page totalsIn CDP Data Visualization, if you use column totals in your Table and Queries         visuals, you can also add a header for column totals.Showing row totalsIn CDP Data Visualization, you can add a totals row to Cross Tabulation     visuals.Specifying totals column font weightCDP Data Visualization allows you to adjust the font style of the totals     column.Specifying maximum number of columnsCDP Data Visualization allows you to limit the number of columns showing for wide     tables.Showing row informationCDP Data Visualization allows you to display the number of rows in your     tables.Showing paginationWith very long tables, scrolling down can become annoying. CDP Data Visualization     allows you to display pagination for your tables, but by default, tables do not display page     numbers.Specifying number of rows per pageShowing header columnIn CDP Data Visualization,  header columns of Table and Queries visuals have a border     by default.Showing header column bordersIn CDP Data Visualization, header columns in Table and Queries visuals have a border by     default.Adjusting table column widthIn CDP Data Visualization, you can manually adjust, either increase or decrease, the     width of columns in a table.Parent topic: Customizing visual style
