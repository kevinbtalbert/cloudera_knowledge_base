Enabling Drill Into data discovery
You can use this feature to drill into data in the context menu of a visual. 

noteThis setting is available only on Bar, Line, Area, and Grouped Bar visual types.
For more information, see Data Discovery Through Context Menus.

To enable the Drill Into
      data discovery option, select the Show Drill Into context menu option
      in the General menu.

If this option is selected, Drill Into appears in the context
        menu. To disable the drill-into behavior, de-select this option.
