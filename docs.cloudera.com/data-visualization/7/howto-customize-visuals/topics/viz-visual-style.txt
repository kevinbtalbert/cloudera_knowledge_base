Customizing visual style
In CDP Data Visualization, Visual Style settings determine the general display of the
        visual. See the following topics for working with different style
        options.
Customizing visual basicsIn CDP Data Visualization, you can modify a visual's background color and also choose     from various font display options for the text that appears in the visual.Customizing table styleIn CDP Data Visualization, you can define various style options for your table     visuals.Customizing table colorsIn CDP Data Visualization, you can redesign the colors that appear in a table     visual.Customizing visual titleIn CDP Data Visualization, you can manage modify the display options of visual     titles.Customizing visual subtitleIn CDP Data Visualization, you can manage modify the display options of visual     subtitles.Customizing visual axes and labelsIn CDP Data Visualization, you can manage several display options for the axes and     lables of your visuals.Customizing tooltipIn CDP Data Visualization, you can manage several visual display options for your     tooltips.Customizing aggregate row styleIn CDP Data Visualization, you can manage several visual display options from the       Aggregate Row Style menu.
