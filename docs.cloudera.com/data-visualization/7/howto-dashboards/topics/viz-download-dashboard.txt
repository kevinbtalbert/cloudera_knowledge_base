Downloading a dashboard
CDP Data Visualization enables you to save the current sheet of a dashboard in
    different file formats.


Downloading dashboard sheets as PNG filesCDP Data Visualization (CDV) enables you to save the current sheet of a dashboard as a     PNG file. The filename is the same as the name of the dashboard, and it has a timestamp     specifying the date and time when the file is created.Downloading dashboards and sheets as PDF filesCDP Data Visualization enables you to save your dashboards, and individuals sheets as     PDF files.
