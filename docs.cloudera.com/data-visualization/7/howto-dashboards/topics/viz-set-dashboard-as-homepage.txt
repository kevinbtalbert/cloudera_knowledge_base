Setting a dashboard as your homepage
In CDP Data Visualization, you can change the home screen to the currently viewed or
    edited dashboard.
You need  edit privileges to perform this task.

Click the supplemental menu in the top menu bar and select Set as
            Homepage.




A message confirming the success of setting the dashboard appears briefly.


Click HOME on the main navigation bar.

The dashboard that you have specified as homage is displayed.
