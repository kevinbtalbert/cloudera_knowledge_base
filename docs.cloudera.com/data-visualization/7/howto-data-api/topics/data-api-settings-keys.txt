Using API keys
When you do not login directly into the visual layer of Cloudera Data Visualization, you must obtain an API key and implement it when
    using Data APIs.
Follow the instructions shown in Creating new API keys.

After obtaining an API Key, you can use it to authenticate the user to access the
          dataset, and invoke the data layer of Cloudera Data Visualization at the
          following endpoint:

http(s)://server:port/arc/api/data


Proceed with using the Data API interface.
For more information, see Example of Data API usage.
