Deleting custom date ranges

To delete set of custom date ranges, follow these steps:

In the Manage Custom Dateranges interface, click
            Trashcan icon next to the set of date ranges to delete.
We are deleting My Date Range Set - Gross Control.





In the Delete Confirmation window modal, type DELETE, and
          then click the Delete button. 







In the Manage Custom Dateranges interface, notice that the list no
        longer has the deleted set.
