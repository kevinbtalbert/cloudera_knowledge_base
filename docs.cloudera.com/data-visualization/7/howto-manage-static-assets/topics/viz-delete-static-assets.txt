Deleting an existing asset


To delete an existing asset, follow these steps.


Click the Gear icon on the main navigation bar to open the
            Administration menu.

Click Static Assets.






In the Manage Images, CSS, and JS Assets interface, click the
            Trashcan icon that corresponds to the asset.






The Delete Confirmation modal, type 'DELETE' and click
            DELETE.
