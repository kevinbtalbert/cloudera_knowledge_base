Assigning multiple roles to multiple users
You can add multiple users to multiple role at the same time in Cloudera Data
    Visualization.
You must have administrative privileges to perform this
      action.

To assign multiple users to existing roles, follow these steps:


Click the Gear icon on the main navigation bar to open the
            Administration menu.

Click Users & Groups.




The Manage Users & Groups  interface appears, open on the
              Users tab.


Select the users that you want to assign to roles. In this example the following users
          are used: CArep1, CArep2, NVrep1, ORrep1, and
          WArep1.






Click ADD TO ROLES.






In the drop-down menu, select the roles you plan to assign to the selected users, and
          click SAVE.

In this example, Test Role 1 and View Only are used.






Note the changes to the information in the Users interface.
