Deleting several user accounts
You can delete user accounts in Cloudera Data Visualization (CDV).
You must have administrative privileges to perform this
      action.

The following steps demonstrate how to delete several CDV user accounts at the same
        time.


Click the Gear icon on the main navigation bar to open the
            Administration menu.

Click Users & Groups.




The Manage Users & Groups  interface appears, open on the
              Users tab.


In the list of users, click the checkmarks next to the usernames you plan to
          delete.

Click DELET.






In the confirmation modal window, click DELETE.
