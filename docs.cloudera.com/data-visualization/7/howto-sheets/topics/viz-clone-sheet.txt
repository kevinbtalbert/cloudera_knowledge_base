Cloning a sheet
CDP Data Visualization enables you to clone a sheet in a dashboard.
You can clone internal and external sheets.

Cloning an internal sheet in a dashboardIn CDP Data Visualization, you can clone sheets within a dashboard.Cloning an external sheet in a dashboardIn CDP Data Visualization, you can clone a sheet from another dashboard, and import it     into your current dashboard.
