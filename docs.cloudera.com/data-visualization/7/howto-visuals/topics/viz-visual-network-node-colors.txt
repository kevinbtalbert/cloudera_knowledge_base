CDP Data Visualization provides many color palette options for nodes in a Network
    visual.

To get to the options in the Colors menu, follow these steps:

On the right side of  the Visual Designer, click the
            Style menu.






 Open the Node Colors menu and choose a color palette. select
          one of the following options:



You can select Reverse color palette of visuals if you want
                to change the order of the colors.


You can click on an individual color swatch in the palette to change the first
                color within palette.


You can select Fix colors for specified dimension
                values.


You can add dimension values.
