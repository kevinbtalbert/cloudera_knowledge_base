Changing result visual type through explore


In the Search Results menu, select the visual you would
                    like to change, and click the Explore icon.




A modal window opens, where you can preview the different visuals. The top
                        left corner of each tile indicates the visual type.





Select one of the visuals.


The new visual type replaces the one originally generated by the search.
