Enumerated values
Cloudera Data Visualization uses standard enumerated types to
        represent objects, actions, and states, such as ComponentType,
            ActionType, and QueryState.
ComponentType
Component type for ArcViz can be one of the following:

APP
DATASET
DATACONNECTION
USER
APPGROUP
CUSTOMSTYLE
CUSTOMCOLOR
SNAPSHOT
THUMBNAIL
STATICASSET
ROLE
CUSTOMEMAIL
CUSTOMDATERANGE
WORKSPACE
SEARCH
RAW_QUERY


ActionType
Action type can be one of the following:

CREATE
READ
UPDATE
DELETE
UNKNOWN


QueryState
Query state can be one of the following:

STARTED
COMPLETE
QUEUED
CANCELED
ABANDONED
