Setting segments


To specify segments in the dataset, you can save information from the
          Filters shelf of a visual as a segment. Follow the steps outlined
        in our article Creating Segments from Filter Definitions. 
Alternatively, you can specify the filters in the Dataset Segments
        interface, as described in Creating Segments.
