Highlighting elements by hovering over legend
 A visual may become very crowded, making it hard to understand what graphical
        elements represent. Cloudera Data Visualization makes it easy to find a particular series by
        making legends not only explanatory, but also interactive.

You can find details of a particular element in the legend by hovering over it.


Open any visual inside a dashboard.

Make sure that the visual displays the legend.For more information, see Adding and removing legend.
Hover on one of the options in the legend to highlight only the data for that
                    species.

This example shows versicolor.
