Creating a dataset from a data extract
Learn how to create a second dataset built on the extracted data, which is used as the
    dataset for building visuals.

Click DATA on the main navigation bar.
The Data view appears, open on the
            Datasets tab.

Click Data Extracts.

Find the data extract that you want to use for the new dataset.

Click CREATE DATASET.






Add a name in the Dataset title field and click
            CREATE.

noteAfter the dataset becomes associated with a data extract, a
            new tag appears in the Datasets list showing Extract
              Target.
