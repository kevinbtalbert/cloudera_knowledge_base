Connection type
This is the name of the database that hosts the data that appears in the form
      DataConnection.This feature is ideal for enterprise environments, with
    dashboards developed on test clusters, and then deployed to a production
    environment.
This assumes that the new connection has, at the minimum, the

      these steps.
