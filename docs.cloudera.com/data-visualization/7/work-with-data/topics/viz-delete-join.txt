Deleting a join
Learn how you can remove an existing join between tables.
Navigate to the Data Model page of the dataset.Click Edit Data Model to edit the data model.

Click the (link) icon that represents the connection that
          must be changed.The Join Details modal window
          appears.Click Delete Join.
In our example, we deleted the cancellation_code connection. Notice
            that this table no longer appears in the Data Model.

[Optional] To revert this change prior to saving, click
          Undo.Click Save.
