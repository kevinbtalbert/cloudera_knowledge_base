Editing dataset fields
You can edit the fields of a dataset by using the Edit button on the dataset field
    tile, next to the drop-down icon.

On the main navigation bar, click DATA.

The Data view appears, open on the Datasets tab.


In the left navigation menu, choose a connection.

In the Datasets area, select  the dataset you want to
          modify.

In the Dataset Detail menu, select
          Fields.






In the Fields interface, select EDIT
            FIELDS.




This will add the (pencil) (edit) button to each dataset field
            tiles.





Click the edit button for the field you want to update.
The Edit Field Parameters modal window appears.

Make your changes and click APPLY.
