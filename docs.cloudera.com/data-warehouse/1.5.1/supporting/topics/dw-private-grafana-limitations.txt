Limitations of Grafana in Cloudera Data Warehouse Private Cloud
Learn about the Grafana capabilities in Cloudera Data Warehouse (CDW) that Cloudera does
  not support. Grafana in CDW is intended for use by cluster operations professionals who are
  familiar with monitoring tools, interpreting metrics, and performing maintenance.
Supported features

Viewing and organizing Grafana dashboards.


Unsupported features

Storing metrics longer than 15 days.
