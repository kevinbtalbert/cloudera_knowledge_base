Creating a custom dashboard
 You can create a custom dashboard in a few steps.

Log into Grafana, and in the Welcome screen, click + > Create





New dashboard appears.



At the top right side of the edit panel, click  to add variables,
                    permissions, and names to the new dashboard.

Click Add a new panel.




On the right side of the New edit panel, click  to add variables
                    and permissions to the new dashboard.






Edit the panel and rename it test1.




Click Apply.
Test1 is added to the dashboard.

Edit the panel, rename it test2, and click
                        Apply.
Test2 is added to the dashboard with Test1.
