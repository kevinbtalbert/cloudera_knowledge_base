Monitoring Impala statestore
 You can monitor Impala statestore from Grafana.

In the list of Impala dashboards, and select Impala - Statestore.

Scroll down, and view live backends, topic size, and heartbeat duration.






Scroll down, and view priority and non-priority topic update duration,
                    container memory usage, statestore memory, and cpu usage.
