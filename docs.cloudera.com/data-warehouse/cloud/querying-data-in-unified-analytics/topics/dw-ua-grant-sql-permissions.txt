Granting permissions to run SQL queries
You must grant Hadoop SQL Ranger permissions to your users so they can query tables. 

In Cloudera Data Warehouse, click  in your Database Catalog and
                    click Open Ranger.






In Ranger Service Manager, click
                 Hadoop SQL.

Select the all - url policy.

In Allow Conditions, click Edit Policy, select users, and add
                 permissions such as Create, Alter, Drop, and Select.

Save the settings.
