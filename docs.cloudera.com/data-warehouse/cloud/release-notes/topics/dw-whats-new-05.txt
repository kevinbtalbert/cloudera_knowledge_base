March 13, 2020
This release of the Cloudera Data Warehouse service on CDP Public Cloud introduces the following new features and
    improvements:
Azure support for Data Warehouse service
This release of Data Warehouse service now supports Microsoft Azure as a public cloud
        provider. For more details about this added support, see Azure support for Data Warehouse service.

Impala Virtual Warehouse improvements
Impala Virtual Warehouses now include the following improvements:

Improved resiliency for client connections.
Added more health checks to more accurately indicate the start of a Virtual Warehouse.
         This helps with query timeouts due to provisioning lags.
