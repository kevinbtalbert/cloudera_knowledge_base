This release of the Cloudera Data Warehouse (CDW) service on CDP Public Cloud
    introduces the new features and improvements that are described in this topic.
Provisioning performance improvements for Virtual Warehouses
The provisioning time for Hive and Impala Virtual Warehouses has been significantly reduced
      for many cases. Generally, provisioning time has been reduced from 1.5 to 4 times over the
      previous time required for Virtual Warehouse provisioning.
