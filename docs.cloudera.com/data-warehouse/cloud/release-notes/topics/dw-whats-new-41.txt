Apache Iceberg V2
This release introduces the general availability of ACID transactions with Iceberg v2 tables from
    Impala in Cloudera Data Platform (CDP). Apache Iceberg ACID transactions run within some of the key data services in the Cloudera Data
    Platform (CDP) public cloud (AWS and Azure), including Cloudera Data Warehouse (CDW).
The following features are included in this release:
Impala reads of Iceberg V2 tables (row-level deletes or updates)
Materialized views of Iceberg tables
Iceberg V2 documentation.
