NiFi component documentation embedded in Flow Designer
The standard NiFi component documentation is directly available in Flow
  Designer.
To access the component documentation, right-click on a component in the
    Canvas and select  Documentation.
noteThe embedded component documentation is retrieved from Apache NiFi and
   may contain information that is not valid in the context of CDF Flow Designer.
