Restarting a test session
Learn about restarting an inactive test session.


Make sure that you have DFDeveloper permission to perform this task. For
                    information on account and resource roles, see DataFlow
                    Authorization.
You have an inactive test session.


Optional: 
You can optionally update the configuration of your NiFi development sandbox
                    before restarting it. Click Flow Options then click
                         Edit Configuration under Test
                        Session.

You have two options to restart an inactive test session:

Click the toggle next to the  Test Session Not Active
                        notification.
Click Flow Options then click  Restart under Test
                            Session.
