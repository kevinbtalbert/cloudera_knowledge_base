Running Python scripts in CDF-PC flow deployments
Relying on Python scripts to perform data transformations within data flows is a common
  pattern for NiFi users. CDF-PC flow deployments come with Python 3.9 and the following custom
  pre-installed packages: requests, urllib3. You can design your data flows to use the pre-installed
  Python runtime as well as install additional custom packages which you might require.
