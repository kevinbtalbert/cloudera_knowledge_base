This release of  Cloudera DataFlow (CDF) on CDP Public Cloud introduces these
  changes.
Fixed Issues

      the associated CDP Environment had been deleted.CDPDFX-3327
Fixed a bug that could result in duplicate system metrics being displayed in the
      Environment details view.CDPDFX-3318
DataFlow now uses the single api.us-west-1.cdp.cloudera.com endpoint for
      Control Plane API access.CDPDFX-3288
