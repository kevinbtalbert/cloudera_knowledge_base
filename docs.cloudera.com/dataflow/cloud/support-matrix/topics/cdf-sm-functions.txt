Functions
AWS Lambda, Google Cloud Functions, and Azure Functions are supported for Cloudera
    DataFlow Functions. You can use any trigger  available in these services when working with
    DataFlow Functions.
All processors and controller services supported in the latest version of the
      Cloudera DataFlow for Public Cloud (CDF-PC)  and Cloudera Flow Management (CFM) are supported
      with DataFlow Functions.
