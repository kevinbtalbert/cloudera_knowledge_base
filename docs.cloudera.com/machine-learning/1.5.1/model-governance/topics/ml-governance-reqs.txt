ML Governance Requirements
You must ensure that the following requirements are satisfied in order to enable ML
  Governance on Private Cloud.
The following services on CDP must be enabled: 
Kafka
Ranger
Solr
Atlas

On Cloudera Manager (CM), ensure that the following are enabled in the base cluster for
   Cloudera Manager: 
Auto-TLS
Kerberos (either MIT or AD)
