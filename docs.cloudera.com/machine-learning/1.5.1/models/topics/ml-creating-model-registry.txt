Creating a Model Registry
Before you can start using Model Registry you must create a model registry for your
  environment. 

From the Cloudera application, click Cloudera Machine
     Learning.

Click Model Registry in the left navigation pane.

Click Create Model Registries.
CML displays the Create Model Registry dialog box. 


Select your environment from the Environment Name field.

Enter the information for your S3 keys, then click Create.
You can now use Model Registry to register, track, and deploy your models.
