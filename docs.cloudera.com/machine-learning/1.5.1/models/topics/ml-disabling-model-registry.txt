Disabling Model Registry
 By default, Model Registry is enabled in CML. You can disable Model Registry if you do
  not want to use this feature.

Click Site Administration in the left navigation pane. 

Click Settings to display the Setting
     Page.

Under the Feature Flags section, uncheck the Enable Model Registry
     checkbox.
