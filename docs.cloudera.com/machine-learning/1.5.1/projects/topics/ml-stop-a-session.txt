Stop a Session
This topic demonstrates how to stop a session to free up resources
    for other users when you are finished.When you are done with the session, click Stop
        in the menu bar above the console, or use code to exit by typing the
        following command:
R
quit()
Python
exit
Scala
quit()Sessions automatically stop after an hour of inactivity.
