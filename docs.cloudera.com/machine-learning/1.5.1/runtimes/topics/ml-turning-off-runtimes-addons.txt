Turning off ML Runtimes Addons
ML Runtimes Addons is turned on by default. However, you can disable ML Runtimes
    Addons.

Select Site Administration > Settings.

Under Feature Flags, uncheck the checkbox next to
            Allow users to Run ML Runtimes Addons.
