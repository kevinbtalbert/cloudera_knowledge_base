Downloading diagnostic bundles for a workspace
Learn how to manage and download diagnostic bundles.
The CDP platform provides various services for managing and downloading diagnostic bundles.
You can download diagnostic bundles from the Cloudera Machine Learning workspace. For more
      information, see Options for generating the CDP Private Cloud diagnostic data.
You can also send usage and diagnostic data from Cloudera Manager. For more information, see:
        Sending Usage and Diagnostic Data to Cloudera.
