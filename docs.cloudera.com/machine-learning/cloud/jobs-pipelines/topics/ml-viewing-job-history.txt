Viewing Job History
This topics shows you how to view the history for jobs run within
    a project.Navigate to the project where the job was created. Click Jobs.Select the relevant job.Click the History tab. You will see a list of
          all the job runs with some basic information such as who created the
          job, run duration, and status. Click individual runs to see the
          session output for each run.
