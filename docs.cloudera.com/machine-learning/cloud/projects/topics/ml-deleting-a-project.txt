Deleting a Project
This topic demonstrates how to delete a project. 

importantDeleting a project is an irreversible action. All
        files, data, and history related to the project will be lost. This
        includes any jobs, sessions or models you created within the project.
      

Go to the project Overview
        page.On the left sidebar, click
          Settings.Go to the Delete Project.Click Delete Project and click
            OK to confirm.
