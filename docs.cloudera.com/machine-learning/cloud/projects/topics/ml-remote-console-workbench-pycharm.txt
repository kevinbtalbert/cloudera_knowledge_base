Configure PyCharm to use Cloudera Machine Learning as the Remote Console


In your project, go to Settings and search for
            Project Interpreter.
 Depending on your operating system, Settings may be called
            Preferences. 

Click the gear icon and select Show All.

Select the Remote Python editor that you added, which is connected to the Cloudera Machine Learning deployment.

Add the following interpreter path by clicking on the folder icon:

/usr/local/bin/python2.7/site-packages
