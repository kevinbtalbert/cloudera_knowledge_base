Disable or Deprecate Runtime Addons
Disable or Deprecate a Spark Runtime addon.
You can disable or deprecate any Spark Runtime addon from the
      Runtime/Engine tab of Site Administration.

Select Site Administration in the left Navigation bar.

Select the Runtime/Engine tab.

Select Disabled or Deprecated from
            Actions next to any Spark addon.
noteYou can
            also return the status to Available using
              Actions.
