Project Admins: Enable the CUDA Engine for your Project
You can make the CUDA-capable engine the default engine for workloads within a
        particular project.
You must be a Project administrator to specify the default
            engine used for workloads within a particular project.

Navigate to your project's Overview page.

Click Settings.

Go to the Engines tab.

Under Engine Image, select the CUDA-capable engine image from the
                    dropdown.
