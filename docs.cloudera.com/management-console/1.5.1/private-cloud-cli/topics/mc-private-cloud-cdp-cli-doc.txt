CLI reference
The CLI reference documentation is available on Cloudera GitHub. It provides the list of
        subcommands and its options for managing your CDP services.
See: CDP CLI Reference.
importantIn 1.4.1, you can use the CDP CLI only for configuring and managing the Data Warehouse service. The available sub-commands for Cloudera Data Warehouse are common for Public and Private Clouds. See the "Form Factors" section to make sure that the command is supported in Private Cloud environments.
