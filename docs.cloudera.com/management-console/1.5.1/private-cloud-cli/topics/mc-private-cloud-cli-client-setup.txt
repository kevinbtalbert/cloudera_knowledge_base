CLI client setup
You must set up the CLI client to use the various available commands. To set up the CDP
    client, you must install the client, generate an API access key, and configure the client with
    the access key.
Installing CDP clientYou can install the CDP client through pip on Linux, macOS, or Windows. The CDP client     works with Python version 3.6 or later.Generating an API access keyA CDP user account (a user or a machine user) must have API access credentials to     access CDP services.Generate an access key for another user or machine userOnly administrator can generate API access keys for other users, including machine         users.Configuring CDP client with the API access keyAfter you generate the access key, go to the command line and run a specific set of     commands to configure the CDP client.
