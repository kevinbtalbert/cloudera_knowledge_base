Working with machine users
For programmatic access to CDP Private Cloud, you must create a machine user. After
        creating the user, you must also generate an API access key.
Creating a machine userYou must create a machine user for programmatic access to CDP Private Cloud.Generating an API access keyA CDP user account (a user or a machine user) must have API access credentials to     access CDP services.Deleting a machine userDeleting a machine user also removes the resources associated with the         user.
