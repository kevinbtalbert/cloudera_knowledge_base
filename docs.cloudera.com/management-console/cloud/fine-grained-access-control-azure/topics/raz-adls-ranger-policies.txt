Ranger policies for RAZ-enabled Azure environment
Ranger enables you to configure resource-based services for Hadoop components and add
        access policies to those services. Ranger includes a set of preloaded resource-based
        services and policies which you can use in a RAZ-enabled Azure environment. 
For more information, see Preloaded resource-based services and
            policies
