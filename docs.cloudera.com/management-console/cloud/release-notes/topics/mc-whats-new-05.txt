This release of the Management Console service introduces the following new
  features:
Creating environments in Azure
CDP supports creating environments in Azure. 
Register your Azure account and then launch Data Hub clusters, Data Warehouse
    clusters, and Machine Learning workspaces within the Azure environment and determine
    which users have access to which resources. These clusters are attached to a Data Lake
    that runs within the environment and provides security and governance for all attached
    clusters.
