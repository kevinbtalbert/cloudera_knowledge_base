This release of the Management Console service introduces the following new
  features:
SSH key management
A Power User can add and delete SSH keys for all users and users can add and delete their own
        SSH keys. For more information, see Managing SSH keys.

Choose Data Lake version at deployment
When deploying an environment, you can now select the Data Lake version. See updated
        instructions for Register an AWS environment and Register an Azure environment.. 

Edit parameters of a previously deployed environment
You can edit the following environment components:


Add new subnets: See Add subnets to an environment.
Add new security groups: See Add security groups to an environment.
Add a new root SSH key: See Add root SSH key to an environment.
