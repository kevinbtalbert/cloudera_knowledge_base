March 7, 2022
This release of the Management Console service introduces the following
  changes:
Public certificate auto-renewal
Most public (Let's Encrypt-issued) certificates for Data Lake and Data Hub clusters will
        now auto-renew without intervention from a user. For more information, refer to Managing Certificates.

Data Lake recipes
Support for attaching/detaching recipes on a Data Lake cluster is now available through
        both the CDP UI and CDP CLI. For more information see  Recipes.
