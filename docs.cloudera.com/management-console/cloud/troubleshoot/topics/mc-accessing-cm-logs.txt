Accessing Cloudera Manager logs
CDP uses Cloudera Manager to orchestrate the installation of Data Lake components. Each
  instance in the cluster runs an Cloudera Manager agent which connects to the Cloudera Manager
  server. Cloudera Manager server is declared by the user during the cluster installation
  wizard.
