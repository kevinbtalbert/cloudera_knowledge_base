Obtain CDP tenant ID
Your CDP tenant ID can be found in the user and machine user profile pages, just below
  the resource CRN.
You may need to obtain your CDP tenant ID, for example when working with Cloudera support. Use
   the following steps to find your CDP tenant ID.
Steps


In CDP web interface, navigate to Management Console > User Management.
Find your user and click on it to navigate to details.
Your Tenant ID is listed among other user details.
