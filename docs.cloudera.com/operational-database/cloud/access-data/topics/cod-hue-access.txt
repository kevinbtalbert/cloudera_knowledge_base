Cloudera Operational Database Hue access
Hue is a web-based interactive SQL editor that enables you to interact with data
        stored in Cloudera Operational Database (COD). You can access the Hue user interface from the COD
        web user interface to create and browse HBase tables. 

Click Databases, and then select a database from the
                    list.

Click Hue.







You can use Hue to quickly browse large tables, create new tables, add data, modify
                existing cells, and also filter data using the auto-complete search.
