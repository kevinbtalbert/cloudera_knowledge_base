Cloudera Operational Database supported programming languages for application
        development
Cloudera Operational Database (COD) enables you to build applications using various
        programming languages. You can choose a programming language to build applications that can
        access data stored in your COD.
List of supported programming languages to access data stored in your COD are as
            follows:


Apache Groovy 


C 


C++ 

Go
Java

Jython 


Python 


PHP


Scala 


You can also use REST for API
            support.
