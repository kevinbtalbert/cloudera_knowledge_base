This release of the Replication Manager service introduces the following new
        features.
Replication Manager introduces HBase replication policy which supports the following use
            cases:

From CDP Private Cloud Base cluster to Data Hub cluster.
From CDH cluster to Data Hub cluster.
From CDH cluster to COD cluster.
From COD cluster to COD cluster.importantHBase replication policy is a technical preview feature. If you
                    require access, contact your Cloudera account team.
