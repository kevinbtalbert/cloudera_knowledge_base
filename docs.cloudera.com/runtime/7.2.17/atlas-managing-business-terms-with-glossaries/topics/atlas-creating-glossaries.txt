Creating glossaries
Create a new glossary when you need to create a new context for
    terms.
Typically you would want to create new glossaries when terms are used by
      significantly different audiences or when the same term has significantly
      different meanings depending on the context. You can create any number of
      glossaries; glossary names can include letters, numbers, spaces, and
      underscores.
