Creating classifications
Create classifications from the tab in the left navigation bar.
Create classifications from the Classifications tab in the left
      navigation pane. 



Names can be 255 characters and contain ASCII characters and spaces. 
        Descriptions can be 1024 characters and contain any characters.
When creating a classification, you can choose an existing
        classification to reuse its attribute definitions.
