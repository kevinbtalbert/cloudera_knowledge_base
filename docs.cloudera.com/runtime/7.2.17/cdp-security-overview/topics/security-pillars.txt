Pillars of Security
To ensure the security of your data, Cloudera follows the Pillars of Security:
  authentication, authorization, encryption, and identity management.
These pillars work together to support our platform.


AuthenticationAuthentication is a basic security requirement in a digital environment. Authorization Authorization is a basic security requirement in a digital environment.EncryptionEncryption increases the level of security in a digital environment.Identity ManagementIdentity management is how user and group identities are centrally stored.
