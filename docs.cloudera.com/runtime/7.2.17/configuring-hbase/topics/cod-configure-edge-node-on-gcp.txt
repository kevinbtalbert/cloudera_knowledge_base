Configuring edge node on GCP
You must configure an edge node on GCP to run applications. 


From the Cloudera Management Console, click Data Hub Clusters
Click Create Data Hub.
In the Selected Environment with running Data Lake drop-down list,
     select the same environment used by your COD instance.
Select the Cluster Definition.
In the Cluster Definition drop-down list, select the
       [***RUNTIME VERSION***] COD Edge Node for [***CLOUD PROVIDER
       NAME***].
In the Cluster Name field, provide a cluster name that you can
     identify later as an edge node of a specific COD instance.
Click Provision Cluster.
