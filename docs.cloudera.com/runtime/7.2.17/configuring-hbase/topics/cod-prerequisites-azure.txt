Prerequisites for configuring an edge node on Azure for COD
You have the ensure that you fulfill all the prerequisites or you will not be able to
  configure an edge node on Azure.
Ensure that you meet the following requirements before starting to configure an edge node on
   Azure:

Permission to log into the Azure account configured with CDP
Permission to launch Virtual Machines in your Azure account
Basic understanding of the Virtual Machines and subnets created for use with CDP
A CDP environment and a Cloudera Operational Database (COD) in the environment in which you want
    the edge node to communicate
Optional: A JSON processing tool such as jq
