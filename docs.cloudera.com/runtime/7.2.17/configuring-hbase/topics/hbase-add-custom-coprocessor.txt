Add a custom coprocessor
 You can add a custom coprocessor to to extend HBase with custom functionality using
    Cloudera Manager.
To configure these properties in Cloudera Manager:

Select the HBase service.

Click the Configuration tab.

Select Scope > All.

Select Category > All.

Type HBase Coprocessor in the Search box.

You can configure the values of the following properties:

HBase Coprocessor Abort on Error (Service-Wide)
HBase Coprocessor Master Classes (Master Default
            Group)
HBase Coprocessor Region Classes (RegionServer Default
            Group)


Enter a Reason for change, and then click Save
            Changes to commit the changes.
