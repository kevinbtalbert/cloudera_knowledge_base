Enable HBase indexing
You can enable HBase indexing using Cloudera Manager.HBase indexing is dependent on the Key-Value Store Indexer service.
Go to the HBase service.
Select Scope > HBASE-1
              (Service Wide)Select
              Category > Backup.Select the Enable Replication and
            Enable Indexing properties. Click Save Changes.
