Monitor the performance of hedged reads
You can monitor the performance of hedged reads using the following metrics provided by
    Hadoop when hedged reads are enabled.
You can monitor the following properties:


hedgedReadOps - the number of hedged reads that have occurred 

hedgeReadOpsWin - the number of times the hedged read returned faster
          than the original read
