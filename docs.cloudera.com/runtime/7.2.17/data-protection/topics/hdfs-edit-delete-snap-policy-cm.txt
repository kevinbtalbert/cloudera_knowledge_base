Edit or delete a snapshot policy
You can use Cloudera Manager to edit or delete existing snapshot
        policies.

Select Backup  > Snapshot Policies in the left navigation bar.
Existing snapshot policies are shown in a table on the Snapshot
                        Policies page.

Click  next to a policy and select Edit
                        Configuration or Delete.

If you want to edit the selected policy, make the required changes and click
                        Save Policy.
