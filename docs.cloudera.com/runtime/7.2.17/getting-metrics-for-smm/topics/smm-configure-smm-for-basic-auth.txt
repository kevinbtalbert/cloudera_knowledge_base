Configuring SMM for basic authentication
To configure Streams Messaging Manager (SMM), you need to configure Prometheus
        username and password, and restart SMM.

Set the following configurations:


Prometheus User (admin)
Prometheus Password (the password you gave in the
                                htpassword tool)



Restart SMM.
