Opening Ranger in Data Hub
You need to navigate to Ranger Admin UI to create a policy for users to access
        Iceberg tables. How you navigate to the Ranger Admin UI differs from one CDP service to
        another, and typically there is more than one path. You learn one way to navigate to the
        Ranger Admin UI from Data Hub.

Log into CDP, and in Cloudera Management Console, click Data Hubs Clusters.




In Data Hubs, select the name of your Data Hub from the list.

In Environment Details, click the link to your Data Lake.
                
For example, click dlscale8-bc8bqz.
