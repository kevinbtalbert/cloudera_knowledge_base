Integrating Schema Registry with Atlas
If you want to view schemas in the Atlas UI, you need to integrate Schema Registry with
    Apache Atlas.
When you integrate Schema Registry with Atlas, the relationship between the schema, topic, and
   all versions of a schema can be explored with ease in Atlas.
