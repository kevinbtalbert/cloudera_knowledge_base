Kafka Connect to Kafka broker security
Learn about the options you have for securing the connection between Kafka Connect and
    Kafka brokers.


Configuring TLS/SSL encryption for the Kafka Connect roleKafka Connect roles inherit the TLS/SSL configuration of the parent Kafka service. If     you are deploying Kafka Connect roles under a Kafka service that already has TLS/SSL enabled,     Cloudera Manager will automatically enable TLS/SSL for Connect as well. If required however, you     can manually enable or disable TLS/SSL.
