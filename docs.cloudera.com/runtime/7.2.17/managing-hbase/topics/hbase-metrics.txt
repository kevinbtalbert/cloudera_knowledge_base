HBase metrics
Use Java Management Extensions (JMX) APIs to collect the metrics exposed by the various
  HBase services. You can also export the metrics in Prometheus format.
You can use various HBase metrics to understand the state of your cluster.

Using JMX for accessing HBase metricsYou can access HBase metrics over Java Management Extensions (JMX) through either the         HBase web interface or by directly accessing the JMX remote agent.Accessing HBase metrics in Prometheus formatYou can access HBase metrics in Prometheus format through the HBase web         interface.
