Editing rack assignments for hosts
You can edit individual host rack assignments from Cloudera Manager.

In Cloudera Manager, go to Hosts > All Hosts.

Select the checkboxes corresponding to the hosts for which you want to specify
                    the racks.

Select Actions for Selected > Assign Rack.

Enter a rack name different from that assigned to the selected hosts, and click
                        Confirm.

Restart the affected services.
