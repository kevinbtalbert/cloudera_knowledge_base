Cloudera Search security aspects
Cloudera Search security covers the following security aspects: 


Securing network communicationCloudera Search supports TLS for encrypting
            communications over a network. 
AuthenticationCloudera Search supports Kerberos and LDAP for authentication.
AuthorizationCloudera Search supports Apache Ranger for authorization.
