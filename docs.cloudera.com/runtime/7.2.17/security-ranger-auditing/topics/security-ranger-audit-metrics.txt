Viewing audit metrics
How to view audit metrics information using the Ranger Admin Web UI.

Metrics provides a high-level view of audit logs as they generate and update in
                Ranger. Ranger captures audit metrics throughput from the following Ranger
                    services:
Atlas
HBase
Hdfs
Hive
Impala
Kafka
Knox
Kudu
NiFi
Schema-registry
Solr
Streams Messaging Manager
Yarn



To view audit metrics, in the Ranger Admin Web UI, click Audit > Metrics.






To view metrics details for a specific service, click
                        Metrics.






To view hourly or daily metrics as a graphic for a specific service, click
                        Metrics Graph.
