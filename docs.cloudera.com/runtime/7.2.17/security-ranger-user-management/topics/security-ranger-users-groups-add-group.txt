Adding a group
How to add a group in Ranger. 

In the Ranger Admin Web UI, select Settings > Users/Groups/Roles > Groups. 
The Groups page appears. 



Click Add New Group.

The Group Create page appears.





Enter a unique name for the group (required) and description (optional), then click
                  Save.
