If you
         have encountered an error and want to search for speecific keywords in the Apache HBase
         log, you can use the log search feature in Cloudera Manager. To search for specific
         keywords in the Apache HBase logs in Cloudera Manager, go to Diagnostics  > Logs . You can search for specific keywords from various sources, services, role
         types, and hosts.
