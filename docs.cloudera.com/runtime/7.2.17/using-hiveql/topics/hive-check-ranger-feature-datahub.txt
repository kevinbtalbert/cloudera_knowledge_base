Check for required Ranger features in Data Hub
Your version of Data Hub might not have the features necessary to create secure
        external tables. Learn how to quickly check for the required Ranger feature. 

In Cloudera Manager, click Clusters > Ranger Admin web UI.

Enter
                    your user name and password, then click Sign In.

Select Hadoop SQL. 

In Hadoop SQL Policies, look for the all - storage-type,
                        storage-url policy.
