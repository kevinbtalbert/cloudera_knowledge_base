Viewing the Impala query execution plan
The query execution plan in Hue provides details on how the query will be executed,
        the operators involved, and other information before the query is submitted to the Impala
        engine.

Log in to the Hue web interface.

Click Jobs from the left assist panel.
The Job Browser page is displayed.

Go to the Impala Queries tab and click on the query for
                    which you want to view the execution plan.
