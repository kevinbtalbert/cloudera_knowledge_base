Accessing the YARN Queue Manager UI
Once you have enabled the Queue Manager UI, you can use it to manage your scheduler
        configuration.




Add the YARN Queue Manager service to your cluster
Configure the YARN Queue Manager dependency




In Cloudera Manager navigate to Clusters >  YARN Queue Manager UI





The Overview tab is opened, displaying a
                    hierarchical view of the queues:



Navigate between the difference tabs to manage queues, configure queue and global
            level properties, and to manage YARN features.
