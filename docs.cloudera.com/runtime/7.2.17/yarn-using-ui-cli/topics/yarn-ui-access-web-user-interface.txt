Accessing the YARN Web User Interface
Access the YARN Web User Interface to monitor clusters, queues, applications, services,
    and flow activities.
You can access the YARN Web User Interface from Cloudera Manager to monitor the
      cluster.

In Cloudera Manager, select the YARN service.

Click the Web UI tab.

Select the ResourceManager Web UI.
The ResourceManager Web UI opens in a new browser window.Note, that in a
            ResourceManager High Availability configuration, if you select the non-active
            ResourceManager the link directs to the web UI of the active
          ResourceManager.
