UI Tools
You can view the YARN configuration and YARN Daemon logs on the
      Tools page.
YARN Configuration
You can see the values of the properties defined in the following configuration files:

Core Configuration: Details of properties defined in the core-default.xml
        file.
YARN Configuration: Details of properties defined in the yarn-default.xml
        file.
MapReduce Configuration: Details of the properties defined in the
          mapred-default.xml file.

YARN Daemon Logs
You can view the list of log files. Click on a log file to view its contents in another tab of your browser.
