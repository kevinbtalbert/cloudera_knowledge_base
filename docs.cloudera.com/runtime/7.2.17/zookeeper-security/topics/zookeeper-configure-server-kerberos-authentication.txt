Configure ZooKeeper server for Kerberos authentication
You can configure the ZooKeeper server for Kerberos authentication in Cloudera
        Manager.

In Cloudera Manager, select the ZooKeeper service.

Click the Configuration tab.

Search for kerberos.

Find the Enable Kerberos Authentication property.

Select the ZooKeeper services that you want to configure for Kerberos
                    authentication.

Click Save Changes.
