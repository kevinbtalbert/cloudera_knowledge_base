A collection of all options and their descriptions for the srm-service
    command line tool.


Option
Description



-h, --help
Shows the help message.


--clusters
Specifies the target clusters for this node.


--config
Specifies the SRM configuration file to use.


--yaml
Specifies the Dropwizard configuration file to use.


--target
Target cluster aliases.
