Auto Action Email Notification Examples
Examples of a Workload XM Auto Actions alert notification email.
The following email notification examples were sent when the
      listed application met the action's criteria and the trigger conditions, which are also
      included in the email notification.Where, as in these examples:
In the Application Details section, the Application ID contains
            a link to the workload application, job, or query.
In the Auto Action Definition section, the Trigger and the
              Criteria definition display both the value and file size type
            that you defined and in brackets the Actual value, in megabytes, that was captured by
            the engine. 
In the Auto Action Results section, the results of the invoked auto action is
            displayed.
