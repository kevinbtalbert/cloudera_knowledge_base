Granting User Access
Workload XM supports two authentication methods for granting user access;
  Local authentication and the Lightweight Directory Access Protocol (LDAP). You configure user
  access to Workload XM in one of these supported authentication methods.


Granting Local AuthenticationGranting user access using local authentication.LDAP Authentication PropertiesGranting user access using the Lightweight Directory Access Protocol (LDAP).
