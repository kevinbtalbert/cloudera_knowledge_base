      Why Cloudera + Microsoft
      






Cloudera Data Platform (CDP), powered by Microsoft Azure, is a secure, global enterprise data cloud that delivers high-performance and faster insights from your data, with a single pane of glass across private, hybrid and public clouds. CDP offers a competitive edge by optimizing speed, flexibility and cost with a modern architecture built on Microsoft Azure offering a secure and compliant hybrid Cloud solution with deep technical data processing and visualization, integration and manageability at global-scale.
Customer benefits

















Data analytics
Multi-functional analytics whenever your data lives













Speed
Gain insights quickly in the cloud













Cost & flexibility
Build burstable data workloads to take advantage of Hybrid Cloud









 














Enterprise security
Enterprise grade security protects data from unauthorized access through out the cluster













Cloud-native
Rapid deployment of CDP on Azure anywhere via AKS













 





We’re pleased to have Cloudera Data Platform join the Azure Marketplace ecosystem. This will open up new opportunities for our mutual customers including an easier, faster path to deployment to meet their growing data engineering needs.
